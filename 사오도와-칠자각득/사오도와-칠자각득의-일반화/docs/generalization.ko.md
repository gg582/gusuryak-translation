# 사오도와 칠자각득의 일반화: "사오도가 칠자각득의 규칙을 따르는가?"

> 본 문서는 하도(사오도) 퍼즐과 칠자각득 퍼즐을 하나의 추상 프레임워크로 통합하고, 사오도가 칠자각득의 구조적 규칙을 어느 정도까지 만족하는지 형식적으로 검증합니다. 전통 용어는 제외하고 현대 조합론·이산수학 언어만 사용합니다.

---

## 1. 동기

두 퍼즐은 다음 공통점을 가집니다.

- 5개의 그룹 또는 클러스터가 존재.
- 숫자가 5로 나눈 잉여에 따라 분류됨.
- 어떤 형태로든 합 불변량이 존재.
- 기하학적 배치가 데이터의 일부임.

그러나 구체적 구조는 다릅니다. 칠자각득은 명시적인 클러스터, 중심-주변 분할, 연결선을 가진 반면, 사오도는 20개의 표식과 5개 색 클래스, 그리고 그룹 간 관계를 가집니다. 본 문서는 이 둘을 포괄하는 일반적 정의를 제시하고, 사오도가 칠자각득의 규칙을 따를 수 있는지 단계별로 판정합니다.

---

## 2. 일반적 정의: 5진 방향-가중 퍼즐

두 퍼즐을 포괄하는 추상 구조를 다음과 같이 정의합니다.

```
Π = (D, M, X, E, K, S, W, Φ)
```

| 기호 | 의미 |
|---|---|
| `D` | 방향 집합. 여기서는 `D = {1, 2, 3, 4, 5}` 또는 `{상, 하, 좌, 우, 중}`으로 동일시 |
| `M` | 표식 또는 원소의 집합 |
| `X` | 기하학적 배치 함수 `X: M → ℝ²` |
| `E` | 연결 관계(변)의 집합, `E ⊆ M × M` |
| `K` | 방향-중심 대응 `K: D → M` 또는 `K: D → ℤ/5ℤ` |
| `S` | 방향별 부차 집합(주변 슬롯) `S: D → 2^M` |
| `W` | 가중 함수 `W: M → ℕ` (숫자 라벨) |
| `Φ` | 불변량 조건의 집합 |

이 정의는 두 퍼즐을 모두 특수한 사례로 포함합니다.

---

## 3. 칠자각득의 규칙: 엄격한 정의

칠자각득이 만족하는 구조적 규칙을 다음과 같이 형식화합니다.

### 규칙 C1. 5개 방향-큰러스터 구조

```
|D| = 5
```

### 규칙 C2. 중심-주변 분할

각 방향 `d ∈ D`에 대해

```
M_d = {K(d)} ∪ S(d)
|S(d)| = 6
|M_d| = 7
```

### 규칙 C3. 중심의 5진 잉여 완전성

```
K: D → ℤ/5ℤ 는 전단사
```

즉 중심 라벨 집합이 `{1, 2, 3, 4, 5}`를 정확히 한 번씩 나타냅니다.

### 규칙 C4. 방향별 합 불변량

```
∃ T ∈ ℕ,  ∀d ∈ D:  W(K(d)) + Σ_{s∈S(d)} W(s) = T
```

칠자각득에서는 `T = 120`입니다.

### 규칙 C5. 명시적 중심-주변 연결선

```
E ⊇ { (K(d), s) | d ∈ D, s ∈ S(d) }
```

즉 변 집합 `E`에 각 방향의 중심과 주변 슬롯을 잇는 변이 포함됩니다.

---

## 4. 사오도를 일반 프레임워크에 대응시키기

사오도의 데이터를 일반 구조 `Π`로 번역하면 다음과 같습니다.

- `D = {1, 2, 3, 4, 5}`: 5개 색 클래스
- `M`: 20개 원형 표식
- `X`: 대칭 십자가형 배치
- `E = ∅`: 원문에 명시된 변이 없음
- `K`: 5개 색 클래스의 대표(중심)를 어떻게 정의할 것인가?
- `S`: 나머지 3개 원소를 해당 색 클래스의 주변 슬롯으로 볼 수 있음
- `W`: 각 표식의 숫자 라벨
- `Φ`: 총합 `Σ W = 210`, 5-컬러링 구조 등

여기서 핵심 질문은 다음과 같습니다.

> 사오도의 각 색 클래스에서 "중심" `K(d)`를 어떻게 선정할 것인가?

자연스러운 선택은 각 색 클래스에서 가장 작은 원소를 중심으로 보는 것입니다.

| 색 클래스 | 중심 `K(d)` | 주변 슬롯 `S(d)` | 합 |
|---|---|---|---|
| 물(1) | 1 | {6, 11, 16} | 34 |
| 불(2) | 2 | {7, 12, 17} | 38 |
| 나무(3) | 3 | {8, 13, 18} | 42 |
| 금속(4) | 4 | {9, 14, 19} | 46 |
| 흙(5) | 5 | {10, 15, 20} | 50 |

이 대응에서 중심 집합은 `{1,2,3,4,5}`이므로 5진 잉여 완전성(C3)은 만족합니다. 그러나 주변 슬롯 수는 3개이지 6개가 아니며, 방향별 합도 동일하지 않습니다.

또 다른 가능성은 2-주기 확장을 고려하여 각 색 클래스를 두 개의 "층"으로 나누는 것입니다.

| 색 클래스 | 층 1 | 층 2 |
|---|---|---|
| 물 | {1, 6} | {11, 16} |
| 불 | {2, 7} | {12, 17} |
| 나무 | {3, 8} | {13, 18} |
| 금속 | {4, 9} | {14, 19} |
| 흙 | {5, 10} | {15, 20} |

이 경우 각 층의 합은:

| 색 클래스 | 층 1 합 | 층 2 합 |
|---|---|---|
| 물 | 7 | 27 |
| 불 | 9 | 29 |
| 나무 | 11 | 31 |
| 금속 | 13 | 33 |
| 흙 | 15 | 35 |

여전히 합 불변량(C4)은 만족하지 않습니다.

## 5. "따름"의 계층적 정의

사오도가 칠자각득의 규칙을 따른다는 말은 여러 수준으로 해석할 수 있습니다. 다음과 같이 단계를 나눕니다.

### 수준 0. 5진 방향 구조 따름

```
|D| = 5 이고, 각 방향에 mod 5 잉여가 할당되어 있다.
```

사오도는 이 수준을 만족합니다.

### 수준 1. 중심-주변 분할 따름

```
각 방향마다 중심 1개와 주변 슬롯 n개로 분할된다.
```

사오도는 중심-주변 분할이 가능하지만, 주변 슬롯 수가 3개로 칠자각득의 6개와 다릅니다.

### 수준 2. 주변 슬롯 수 따름

```
∀d ∈ D: |S(d)| = 6
```

사오도는 만족하지 않습니다. 주변 슬롯이 3개뿐입니다.

### 수준 3. 방향별 합 불변량 따름

```
∃ T, ∀d ∈ D: W(K(d)) + Σ_{s∈S(d)} W(s) = T
```

사오도는 만족하지 않습니다. 각 색 클래스의 합은 34, 38, 42, 46, 50으로 다릅니다.

### 수준 4. 명시적 연결선 따름

```
중심-주변 연결선이 원문 데이터에 포함된다.
```

사오도는 만족하지 않습니다. 원문에 변이 명시되지 않았습니다.

---

## 6. 결론: 사오도는 칠자각득의 규칙을 따르는가?

**엄격한 의미에서는 따르지 않는다.**

사오도는 칠자각득의 핵심 규칙 C2, C4, C5를 만족하지 않습니다. 특히 주변 슬롯 수, 방향별 합 불변량, 명시적 연결선 측면에서 구조가 다릅니다.

**그러나 약한 의미에서는 공통 골격을 공유한다.**

사오도는 5개 방향 구조, 5진 잉여 분류, 중심의 완전 잉여계, 그리고 전역 합 불변량이라는 칠자각득의 기본 골격을 공유합니다. 따라서 사오도는 칠자각득 규칙의 **약화된 변형** 또는 **다른 파라미터 설정**으로 볼 수 있습니다.

형식적으로 다음과 같이 기술할 수 있습니다.

> 사오도는 칠자각득의 "5진 방향-중심 골격"은 따르지만, "6주변-합일치-연결선 구조"는 따르지 않는다.

---

## 7. 통합 일반화: 파라미터화된 퍼즐족

두 퍼즐을 하나의 파라미터화된 족으로 볼 수 있습니다.

```
Π(p, q, T)
```

- `p`: 방향 수(여기서는 5)
- `q`: 각 방향의 주변 슬롯 수
- `T`: 각 방향(큰러스터)의 목표 합

칠자각득은 `Π(5, 6, 120)`입니다.

사오도를 색 클래스별로 본다면 `Π(5, 3, T_d)`이며, 여기서 `T_d`는 방향에 따라 다릅니다. 즉 사오도는 **비균일 파라미터 설정**을 가진 동일 족의 다른 예입니다.

| 퍼즐 | p | q | T |
|---|---|---|---|
| 칠자각득 | 5 | 6 | 120 (모든 방향 동일) |
| 사오도(색 클래스) | 5 | 3 | 34, 38, 42, 46, 50 (방향별 상이) |

---

## 8. 현대 조합론적 문제

### 문제 1. 파라미터족의 존재 조건

주어진 `p, q, T`에 대해 `Π(p, q, T)` 형태의 퍼즐이 존재하기 위한 필요충분조건을 찾아라. 특히 `p = 5`일 때, `q`와 `T`가 자연수 집합 위에서 어떤 제약을 받는지 조사하라.

### 문제 2. 사오도의 균일화

사오도의 20개 표식을 5개 클러스터로 재배치하여 각 클러스터가 7개 원소를 갖고 합이 동일하도록 할 수 있는가? 이때 원래의 5-컬러링과 기하 배치를 얼마나 보존할 수 있는가?

### 문제 3. 방향별 합 불변량의 필요성

칠자각득에서는 방향별 합이 일정하지만, 사오도에서는 그렇지 않다. 이 차이가 퍼즐의 "완성도"나 "난이도"에 어떤 영향을 주는가? 합 불변량이 없을 때 추가로 필요한 조건은 무엇인가?

### 문제 4. 변의 부재와 보완 체계

사오도에는 명시적 연결선이 없다. 변 집합 `E`를 최소한으로 추가하여 칠자각득형 그래프 구조를 만들되, 원문에 없는 가정을 최소화하는 방법을 제시하라.

---

## 9. 요약

- 사오도와 칠자각득은 `Π(p, q, T)`라는 공통 프레임워크의 특수 사례이다.
- 칠자각득은 `Π(5, 6, 120)`이라는 엄격한 구조를 가진다.
- 사오도는 `Π(5, 3, T_d)` 형태의 비균일 구조로, 같은 족에 속하지만 칠자각득의 엄격한 규칙을 따르지 않는다.
- 따라서 "사오도가 칠자각득의 규칙을 따른다"는 말은 **약한 골격 공유**까지만 정당화되며, **강한 구조 동일성** 주장은 거짓이다.

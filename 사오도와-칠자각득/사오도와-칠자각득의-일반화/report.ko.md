# 통합 보고서: 사오도와 칠자각득의 일반화

**작성일**: 2026-07-07

---

## 1. 개요

본 보고서는 하도(사오도) 퍼즐과 칠자각득 퍼즐을 하나의 추상 프레임워크 `Π(p, q, T)`로 통합하고, 사오도가 칠자각득의 구조적 규칙을 어느 정도까지 따르는지 형식적으로 분석한 결과입니다.

---

## 2. 일반 프레임워크

두 퍼즐을 포괄하는 정의:

```
Π = (D, M, X, E, K, S, W, Φ)
```

| 기호 | 의미 |
|---|---|
| `D` | 방향 집합(5개) |
| `M` | 원소/표식 집합 |
| `X` | 기하 배치 |
| `E` | 변 집합 |
| `K` | 방향-중심 대응 |
| `S` | 방향별 주변 슬롯 |
| `W` | 가중 함수(숫자 라벨) |
| `Φ` | 불변량 조건 |

파라미터화된 퍼즐족:

```
Π(p, q, T)
```

- `p`: 방향 수
- `q`: 각 방향의 주변 슬롯 수
- `T`: 각 방향의 목표 합

---

## 3. 두 퍼즐의 파라미터 대응

| 퍼즐 | p | q | T |
|---|---|---|---|
| 칠자각득 | 5 | 6 | 120 (모든 방향 동일) |
| 사오도(색 클래스) | 5 | 3 | 34, 38, 42, 46, 50 (방향별 상이) |

---

## 4. 따름 단계별 판정

| 단계 | 조건 | 사오도 만족 여부 |
|---|---|---|
| 0 | 5개 방향 구조 | 예 |
| 1 | 중심-주변 분할 가능 | 예 (주변 3개) |
| 2 | 주변 슬롯 수 = 6 | 아니오 |
| 3 | 방향별 합 불변량 | 아니오 |
| 4 | 명시적 연결선 | 아니오 |

---

## 5. 결론

- 사오도는 칠자각득의 **약한 골격**(5개 방향, 5진 잉여, 중심 완전 잉여계, 전역 합 불변량)은 공유한다.
- 그러나 **강한 구조**(6개 주변 슬롯, 방향별 합 일치, 명시적 연결선)는 따르지 않는다.
- 따라서 "사오도가 칠자각득의 규칙을 따른다"는 말은 엄격하게는 거짓이며, 동일 족 내 비균일 파라미터 설정으로 보는 것이 적절하다.

## 7. 검증 스크립트 결과

`compare_structures.py` 실행 결과:

- 칠자각득: 방향 수 5, 중심 `{1,2,3,4,5}`, 주변 슬롯 수 6, 모든 클러스터 합 120.
- 사오도: 중심 `{1,2,3,4,5}`로 5진 완전 잉여계는 형성하나, 주변 슬롯 수는 3, 클러스터 합은 34/38/42/46/50로 불일치.
- 결과적으로 사오도는 `Π(5, 3, T_d)` 형태의 비균일 파라미터 설정임이 확인됨.

# Baekjayin-yang-jamo-chakjong (Hundred-Numbers Yin-Yang Mother-Child Intertwining Diagram, 百子陰陽子母錯綜圖)

Order: 10×10

## Normal Magic Square Status

**Not a normal magic square.** It uses the normal value set 1~100 exactly once, but not every row, column, and diagonal sums to the magic constant 505. Eight rows and eight columns sum to 505, but the remaining rows and columns do not.

A valid 10×10 magic square correction is given separately in [`corrected.md`](corrected.md).

## Generation Rule Summary

Baekjayin-yang-jamo-chakjong is a 10×10 matrix that uses each of the natural numbers 1 through 100 **exactly once**. Therefore its value set is the same as that of a normal magic square, but because the row, column, and diagonal sums are not all equal, it is **not a magic square**.

The terms "yin-yang (陰陽)" and "jamo (子母, mother-child)" in the name suggest that the numbers are arranged in complementary pairs (mostly pairs that sum to 101).

## Why Is It Not a Magic Square?

The value set uses each of 1~100 exactly once, but **8 rows sum to 505** while **2 rows (rows 5 and 6) sum to 495 and 515**, respectively. Similarly, some column sums are 525 or 485 instead of 505. Therefore it does not satisfy the definition of a magic square.

However, the fact that 8 rows and 8 columns sum to 505 means this matrix is close to a "near-magic square."

## Yin-Yang / Mother-Child Pair Analysis

Call pairs of numbers from 1~100 that sum to 101 "yin-yang pairs" or "mother-child pairs," e.g., (1,100), (2,99), ..., (50,51).

Investigating the cases where the centrally symmetric positions in this matrix sum to 101, we find that it is not a fully associated structure, but many pairs do have the 101 relationship.

In particular, complementary pairs are clearly visible along the borders, such as between row 1 and column 10, and between column 1 and row 10.

## Estimated Generation Rule

This matrix was likely generated by the following procedure:
1. Group the numbers 1~100 into complementary pairs (sum 101).
2. Arrange them so that most rows and columns sum to 505.
3. Intentionally introduce deviations of ±10 and ±20 in some rows (5, 6) and columns (3, 8).

These deviations are mutually complementary, so the overall sum remains balanced. This is the mathematical expression of "chakjong (錯綜)," a state in which yin and yang are crossed and intertwined.

## Conclusion

Baekjayin-yang-jamo-chakjong uses the normal value set (1~100) but is not a magic square. Instead, it is a special square centered on the concept of complementary "yin-yang / mother-child" pairs. Most rows and columns sum to the magic constant 505, but some rows and columns show intentional deviations.

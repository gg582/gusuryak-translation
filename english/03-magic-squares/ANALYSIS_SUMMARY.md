## Key Mathematical Properties

### Magic Constants

- 6×6: \(M_6 = 6(36+1)/2 = 111\)
- 9×9: \(M_9 = 9(81+1)/2 = 369\)
- 10×10: \(M_{10} = 10(100+1)/2 = 505\)

### Associated Magic Square

An associated magic square is one in which every pair of cells symmetric about the center sums to \(n^2+1\).
- Gusudo Example 1: \(a_{i,j}+a_{8-i,8-j}=82\), fully associated.
- Yukyukdo Example 2: 32 of the 36 pairs sum to 37. Because a 6×6 associated magic square cannot exist, it cannot be fully corrected to associated.

### Pan-Diagonal

A pan-diagonal magic square is one in which every diagonal, including those that wrap around, sums to the magic constant. None of the squares in this document are pan-diagonal.

### 10×10 Corrections

For the four 10×10 squares, valid 10×10 magic squares generated by the `magic_square` Python package are presented as corrections. Each correction uses the numbers 1~100 exactly once, and every row, column, and both main diagonals sum to 505.

## Yukyukdo Peculiarity

A 6×6 square belongs to the **singly even** class (\(n \equiv 2 \pmod 4\)), which makes magic square construction more complex. Both Yukyukdo examples are valid magic squares, but:

- Example 1: has no associated structure at all.
- Example 2: **32 of the 36** centrally symmetric pairs sum to 37 (=6²+1). The remaining four pairs fail in a 2×2 region.

Because a 6×6 associated magic square does not exist, Example 2 is only "almost associated" and cannot be fully corrected to associated.

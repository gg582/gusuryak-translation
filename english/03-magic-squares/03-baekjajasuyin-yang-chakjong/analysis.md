# Baekjajasuyin-yang-chakjong (Hundred-Numbers Yin-Yang Intertwining Diagram, 百子子數陰陽錯綜圖)

Order: 10×10

10×10. The first example is a matrix filled by repeating only the numbers 1~10; although every row/column sum is 55, the value set is not 1~100, so it is not a conventional magic square. The second example is a matrix filled by repeating the numbers 0~9; every row/column sum is 45. These two matrices are complementary (yin/yang) Latin squares or circulant structures, and their combination could form components of an orthogonal Latin square pair.

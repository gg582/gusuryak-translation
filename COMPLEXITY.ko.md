# 계산 복잡도 범위와 증명된 환원

이 문서는 저장소의 구체적인 도안 검증과 일반화된 배치 문제의 복잡도를
분리한다.

## 증명된 명제

**일반 등합 배치 문제**를 다음과 같이 정의한다. 입력은 양의 정수 가중치
`a_1, ..., a_{3m}`, 서로 겹치지 않는 `m`개 그룹, 그룹 크기 3, 목표합 `B`로
이루어진다. 모든 가중치를 정확히 한 번씩 사용하여 각 그룹의 합을 `B`로
만들 수 있는지를 묻는다.

이 문제는 **3-PARTITION**에서 항등 환원되므로 강한 NP-완전이다.

1. 3-PARTITION 입력은 `A = {a_1, ..., a_{3m}}`, `sum(A) = mB`,
   `B/4 < a_i < B/2`를 만족한다.
2. `m`개의 각득 그룹을 만들고 각 그룹 크기를 3, 공통 합을 `B`로 정한다.
   가중치는 그대로 복사한다.
3. 각득 배치가 존재하는 것은 `A`를 합이 `B`인 세 수 묶음 `m`개로 나눌 수
   있는 것과 정확히 동치다. 변환은 항등 함수이므로 다항 시간이다.
4. 그룹 배치는 다항 시간에 검증할 수 있으므로 이 문제는 NP에 속한다.
   따라서 강한 NP-완전성이 성립한다.

3-PARTITION의 강한 NP-완전성은 Garey와 Johnson의 표준 결과다. 이 환원은
임의의 가중치와 가변 그룹을 허용하는 일반화에 대한 것이다.

## 이 환원이 증명하지 않는 것

이 환원만으로 고정된 역사적 그래프, 연속 수열 `1..N`, 특정 구수략 도안의
고정 겹침 구조가 NP-완전이라고 결론 내릴 수는 없다. 이러한 제한 문제는 더
쉬울 수 있으며, 이 저장소는 해당 도안들에 대한 완전한 환원을 제시하지 않는다.
구체적인 도안의 합·범위·중복·그래프 불변량 검증은 직접 검사할 수 있으므로
다항 시간이다.

README의 표기는 다음 원칙으로 정리한다.

- **P**: 주어진 유한 도안을 직접 검증하는 문제
- **증명된 NP-완전**: 위의 임의 가중치·가변 그룹 일반화처럼 환원이 이 저장소에
  명시된 문제
- **미해결 / 이 저장소에서 미증명**: 고정 위상이나 연속 수열을 유지한 변형

## 이 저장소에서 재현한 검사

`python3 complexity/verify_polynomial_checks.py`는 현재 다음을 출력한다.

- 육육도 6×6 원 배열 2개: 정상 마방진, 선합 111;
- 구수도 9×9 원 배열 2개: 정상 마방진, 선합 369;
- 백자 계통 10행 블록 5개: 앞의 4개는 정상 집합 검사 실패, 마지막 자모
  블록은 통과한다. 첫 번째 도안이 두 성분 블록을 가지므로 도안 수보다 블록 수가
  하나 더 많다;
- 한영판의 10×10 교정본 8개: `1..100`을 정확히 한 번씩 사용하고 모든 행·열·
  주대각선의 합이 505;
- 한영판 육고도 기하 테스트: 양쪽 모두 종료 상태 0.

스크립트는 여러 행렬 크기에서 직접 검증 루틴의 실행 시간도 측정한다. 이
측정은 구현의 회귀 검사 자료이지, 복잡도 증명의 대체물이 아니다.

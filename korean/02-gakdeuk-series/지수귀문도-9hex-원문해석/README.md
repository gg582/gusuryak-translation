## 원문

> 六子各得九十三數 — 6개 수의 부분합이 93이다.  
> 九宮共得八百三十七數 — 9개 궁의 합은 모두 837이다.  
> 中眷三宮，三宮爲主則 — 가욱 중앙의 것이 3개 궁을 돌보고, 3개 궁이 주가 된다.  
> 左右十二子 分爲二宮 若爲以 — 좌우 12자가 2개 궁으로 나뉜다.  
> 四正爲四宮，中間六子合爲一宮 — 4개 꼭지 방위가 4개 궁이고, 중간 6개 수를 1개 궁으로 묶는다.  
> 凡九宮化爲十二宮 — 무릇 9개 궁이 12개 궁이 된다.

자세한 병렬 해석은 `origin_text.md`를 본다.

## 作(작)과 用(용)의 분리

| 항목 | 값 |
|---|---:|
| 고유 수 (作) | 30개 (1~30) |
| 구조상 쓰이는 자리 (用) | 54개 (9육각형 × 6정점) |
| 중복 자리 수 | 24개 |
| 고유 수의 합 (T) | 465 |
| 9육각형 중복 포함 총합 (H·S) | 9 × 93 = 837 |
| 중복 가중 합 (D) | 372 |

이것이 원문의 **三十子作， 五十四子用**에 해당한다.

### 정점별 중복 계수

| 중복 계수 | 정점 (원래 번호 = 배치 값) |

| 1회 | 1=1, 3=30, 4=5, 5=25, 7=12, 9=4, 11=19, 20=26, 22=13, 24=11, 26=16, 28=8, 29=28, 30=20 |
| 2회 | 2=14, 6=27, 8=3, 10=2, 19=21, 23=18, 25=15, 27=22 |
| 3회 | 12=9, 13=17, 14=7, 15=24, 16=29, 17=6, 18=10, 21=23 |

## mod 분류

잉여는 1-based로 계산: `r = value % mod`, 나누어떨어지면 `r = mod`.

### mod 2

| 클래스 | 개수 | 합 |
|---|---:|---:|

| r2 (짝수) | 15 | 240 |

## CRT 재구성 (진구소 / Qin Jiushao)

서로소인 mod 쌍을 중국인의 나머지 정리(Chinese Remainder Theorem)로 결합한다.

| 쌍 | 결합 mod | 의미 |
|---|---|---|
| mod 3 × mod 4 | mod 12 | mod 3과 mod 4로부터 mod 12 클래스를 복원 |
| mod 3 × mod 5 | mod 15 | 3방위 구조와 5방위 구조를 연결 |
| mod 4 × mod 5 | mod 20 | 4방위 구조와 5방위 구조를 연결 |

`crt_mod3_times_mod4.png`, `crt_mod3_times_mod5.png`, `crt_mod4_times_mod5.png`에서 결합 클래스의 공간 분포를 볼 수 있다.

## 2·3 기반 변이

- **mod 2**: 가장 작은 소모듈러; 홀짝 및 대칭성을 담당.
- **mod 3**: 3방위 / 9궁 골격에 대응하는 기본 모듈러.
- **mod 4 = 2×2**: mod 2의 한 단계 세분화.
- **mod 6 = 2×3**: 홀짝과 3방위 구조를 동시에 본다.
- **mod 9 = 3×3**: 9궁 체계와 호응하는 3의 제곱.
- **mod 12 = 2²×3**: mod 3과 mod 4의 CRT로 완전히 복원.

이 moduli family는 임의의 수가 아니라 기본 2와 3으로부터 생성되며, 이는 공간 그룹핑이 사도(四道)와 오도(五道)의 상호작용에서 나온 것과 같은 맥락이다.

## 확장 시각화 해석

### 作/用 분리 (`jisu_9hex_multiplicity.png`)

- 30개의 서로 다른 수(作)가 54개의 육각형 자리(用)에 배치된다.
- 중앙으로 갈수록 정점이 여러 육각형에 공유되어 중복 계수가 커진다.
- 중복 계수 3회 정점은 8개로, 모두 중앙 육각형(Hex5)과 그 주변에 집중된다.
- 이는 원문 **三十子作， 五十四子用**을 직접적으로 보여준다.

### mod 2 대칭 (`jisu_9hex_mod2_symmetry.png`)

- 홀수 15개, 짝수 15개로 완벽히 균등하다.
- 좌우 대칭선(x=0)과 상하 대칭선(y=0)을 기준으로 홀수가 비교적 고르게 분포한다.
- 이는 정육각형 격자의 반사 대칭성과 부분합 불변식 사이의 균형을 시각화한다.

### 9궁 → 12궁 재해석 (`jisu_9hex_9to12_palaces.png`)

- 안쪽 파란 원은 9개의 육각형 궁(九宫)을, 바깥쪽 주황 원은 12개의 방위 궁(十二宫)을 나타낸다.
- 원문 **凡九宮化爲十二宮**은 9개 육각형 중심을 12방위(12지) 체계로 재해석할 수 있음을 의미한다.
- 회색 연결선은 각 육각형이 인접한 방위 섹터에 걸쳐 매핑됨을 보여준다.

### 중심 3궁 구조 (`jisu_9hex_center_periphery.png`)

- 세로 중앙열 Hex2, Hex5, Hex8을 빨간색으로 강조한다.
- 이 3개의 육각형은 좌우 6개 육각형을 관장하는 중심 축이다.
- 원문 **中眷三宮， 三宮爲主則**은 이 중심 3궁이 전체 구조의 주(主)가 됨을 나타낸다.

### 그래프 스펙트럼 (`jisu_9hex_adjacency_spectrum.png`)

- 30×30 인접행렬: 9개 육각형의 공유 정점 구조가 희소 패턴으로 드러난다.
- 차수분포: 대부분의 정점이 2차 또는 3차이며, 중앙 일부 정점이 높은 차수를 가진다.
- 고유값 분포: 인접행렬과 Laplacian의 스펙트럼은 그래프의 대칭성과 연결성을 수량화한다.

### 마법 상수 강조 (`jisu_9hex_magic_constant.png`)

- 각 육각형 중앙에 **93**을 표시하고, 구성 값을 함께 적는다.
- 9개 육각형 모두 합이 93임을 한눈에 확인할 수 있다.
- 이는 원문 **六子各得九十三數**의 직접적인 시각화이다.

## 유전적 알고리즘 solver

각득 계열 관점에서 노드 배치의 최적해를 찾거나 예산 내에서 해를 찾지 못함을
알아내기 위한 GA 기반 solver도 함께 제공한다.

| 파일 | 설명 |
|---|---|
| `ga_gakdeuk_solver.py` | 일반 순열 GA에 전체/연속 스왑 지역 탐색을 결합한 solver |
| `paper_hga_htp_solver.py` | GECCO 2003 지수귀문도 하이브리드 GA 구현 |
| `compare_ga_paper.md` | 두 GA 접근법과 참고 논문의 비교 분석 |

실행:

```bash
python ga_gakdeuk_solver.py      # 일반 GA 각득 solver
python paper_hga_htp_solver.py   # 논문의 하이브리드 GA
```

두 solver 모두 클러스터 합의 분산을 적합도로 사용하며, 완벽한 배치를 찾으면
`optimal`, 예산 내에 찾지 못하면 `not_found_within_limit`를 반환한다.

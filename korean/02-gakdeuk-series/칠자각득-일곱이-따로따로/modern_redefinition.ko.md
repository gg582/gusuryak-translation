# 칠자각득(七子各得) 퍼즐의 현대 조합론적 재정의

> 본 문서는 전통 용어를 배제하고, 도식에 나타난 데이터만으로 현대 조합론 및 이산수학 언어를 사용해 퍼즐을 재정의합니다.

---

## 1. 관측된 객체

도식은 다음 요소로 구성됩니다.

- **5개의 클러스터**: 원형 경계 안에 배치된 7개 원소의 집합
- **중심**: 각 클러스터의 중앙에 있는 1개 원소
- **주변 슬롯**: 중심 주위의 6개 원소
- **연결선**: 중심과 각 주변 슬롯을 잇는 선분
- **숫자 라벨**: 각 원소에 부여된 자연수
- **공간 배치**: 5개 클러스터가 십자가 형태로 배치됨
- **합 조건**: 각 클러스터의 7개 원소 합이 120

이 정의에서 원형 경계는 시각적 틀일 뿐, 수학적으로는 클러스터를 구분하는 집합적 구분자입니다. 연결선 `E`는 명시적으로 주어진 변 집합이며, 이는 이전 사오도 퍼즐과 달리 **그래프 구조가 데이터의 일부**임을 의미합니다.

관측된 5개 클러스터는 다음과 같습니다.

| 클러스터 | 중심 | 주변 슬롯 | 합 |
|---|---|---|---|
| C₁ (상) | 2 | 29, 1, 24, 34, 11, 19 | 120 |
| C₂ (좌) | 3 | 6, 33, 23, 13, 34, 8 | 120 |
| C₃ (중) | 5 | 22, 7, 20, 30, 26, 10 | 120 |
| C₄ (우) | 4 | 15, 28, 9, 18, 32, 14 | 120 |
| C₅ (하) | 1 | 35, 16, 21, 24, 6, 17 | 120 |

- 전체 원소 개수 (중복 포함): 35
- 고유한 숫자 값: 31개
- 중복 값: 1, 6, 24, 34 (각 2회)
- 전체 합 (중복 포함): 600 = 5 × 120

---

## 2. 형식적 정의

원문에서 직접 추출할 수 있는 구조는 다음과 같습니다.

```
P = (C, K, S, X, E, I)
```

| 기호 | 의미 | 현대적 용어 |
|---|---|---|
| `C` | 5개 클러스터의 집합 | 클러스터 집합 |
| `K` | 각 클러스터의 중심 라벨 함수 `K: C → ℕ` | 중심 라벨링 |
| `S` | 각 클러스터의 주변 슬롯 라벨 함수 `S: C → ℕ⁶` | 주변 슬롯 라벨링 |
| `X` | 클러스터들의 2차원 기하 배치 | 기하 배치 |
| `E` | 중심-주변 연결선 집합 | 반지름 변 집합 |
| `I` | 각 클러스터의 합 불변량 `Σ(c) = 120` | 국소 합 불변량 |

---

## 3. 5로 나눈 잉여 구조

각 클러스터의 중심은 `{1, 2, 3, 4, 5}`를 정확히 한 번씩 형성합니다. 이는 `ℤ/5ℤ`의 완전 잉여계입니다.

```
K = {1, 2, 3, 4, 5}
```

주변 슬롯의 5로 나눈 잉여 분포는 다음과 같습니다.

| 클러스터 | 중심 잉여 | 주변 잉여 분포 |
|---|---|---|
| C₁ | 2 | 1×2, 4×4 |
| C₂ | 3 | 1×1, 3×4, 4×1 |
| C₃ | 5 | 1×1, 2×2, 5×3 |
| C₄ | 4 | 2×1, 3×2, 4×2, 5×1 |
| C₅ | 1 | 1×3, 2×1, 4×1, 5×1 |

각 클러스터 전체의 잉여 합은 5의 배수가 되며, 이는 총합 120이 5로 나누어떨어지기 때문입니다.

---

## 4. 기하학적 배치

5개 클러스터는 다음 좌표에 배치됩니다.

```
        C₁(0, 3.3)
            |
C₂(-3.3, 0) — C₃(0,0) — C₄(3.3, 0)
            |
        C₅(0, -3.3)
```

각 클러스터 낶부에서 6개 주변 슬롯은 중심을 둘러싼 정육각형에 가깝게 배치됩니다. 따라서 전체 구조는 **5개의 7-점 별 모양 배치가 십자가로 배치된 별자리**로 볼 수 있습니다.

---

## 5. "사오도의 규칙"에 대한 현대적 재해석

원문 주석에서 "사오도의 규칙을 따른다"고 했습니다. 이를 현대 언어로 옮기면 다음과 같습니다.

1. **5개의 방향-중심 대응**: 5개 클러스터는 5개 방향(상/하/좌/우/중)에 놓이고, 각 방향의 중심은 `1,2,3,4,5` 중 하나입니다.
2. **중심-주변 분할**: 각 클러스터는 중심 1개와 주변 6개로 분할됩니다. 이는 "중심 + 6방향"의 구조입니다.
3. **방향별 합 불변량**: 각 방향(큰러스터)의 총합이 동일(120)합니다.
4. **5로 나눈 잉여 완전성**: 중심들이 mod 5의 모든 잉여를 정확히 한 번씩 나타냅니다.

이를 일반화하면, 이 도식은 **방향 집합 `D` 위에서 정의된 가중 함수**로 볼 수 있습니다.

```
D = {상, 좌, 중, 우, 하}
K: D → ℤ/5ℤ   (전단사)
S: D → (ℕ⁶)
Σ_{d∈D} (K(d) + ΣS(d)) = 600
∀d∈D: K(d) + ΣS(d) = 120
```

## 6. 현대 조합론적 문제 추출

### 문제 1. 합 불변량 방정식

각 클러스터의 합이 120이 되도록 6개 주변 수를 선택하는 방법의 수를 계산하라. 단, 주변 수의 범위와 중복 조건을 명시하라.

### 문제 2. 5-컬러링

각 클러스터의 중심과 주변 슬롯들이 5로 나눈 잉여를 어떻게 분배하는지 조사하라. 각 클러스터의 잉여 합이 0 (mod 5)이 되는 이유를 설명하라.

### 문제 3. 중복 구조

1, 6, 24, 34가 두 클러스터에 걸쳐 등장한다. 이 중복이 우연인지, 아니면 클러스터 간 관계를 암시하는지 분석하라.

### 문제 4. 방향 그래프

5개 클러스터를 정점으로 하고, 중복된 숫자나 공간적 인접성을 변으로 하는 그래프를 정의하라. 이 그래프의 연결성과 대칭성을 조사하라.

---

## 7. 강점과 약점

### 강점

1. 명확한 합 불변량(120)이 존재.
2. 중심이 5로 나눈 완전 잉여계를 형성.
3. 기하학적 배치(십자가 + 정육각형)가 규칙적.
4. 중심-주변 연결선이 명시적으로 주어져 있어 그래프 구조가 분명.

### 약점

1. 주변 수 선택의 규칙이 명시되지 않음.
2. 중복된 값(1, 6, 24, 34)의 의미 불명.
3. 120이라는 합의 출처(왜 120인가?)가 설명되지 않음.
4. 클러스터 간 관계(변, 연산 등)가 정의되지 않음.

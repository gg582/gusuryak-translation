# 지수귀문도와 지수용육도의 관계

## 1. 개요

**지수용육도(地數用六圖)**와 **지수귀문도(地數龜文圖)**는 모두 조선 후기 수학서 《구수략(九數略)》 계열에 전하는 도상 퍼즐입니다. 둘 다 육각형 꼭짓점에 수를 배치하여 **각 육각형을 이루는 여섯 수의 합이 모두 같도록** 만드는 규칙을 따릅니다.

| 항목 | 지수용육도 | 지수귀문도(대표 예) |
|---|---|---|
| 육각형 수 | 5개 | 3, 4, 6, 7, 9, 10, 19개 등 |
| 노드 수 M | 20 | 13, 16, 22, 24, 30, 33, 54 등 |
| 사용 수 | 1–20 | 1–M |
| 마법 상수 S | 63 | 34–50, 40–62, 57–71, 65–85, 77–109, … |
| 핵심 규칙 | 모든 육각형의 6개 수 합 동일 | 모든 육각형의 6개 수 합 동일 |

지수귀문도는 지수용육도의 **구조적 확장**으로 볼 수 있습니다. 지수용육도가 "5개 육각형, 20개 노드"라는 특수한 형태라면, 지수귀문도는 **같은 합 불변량 규칙을 여러 크기의 육각형 타일링에 적용한 일반화**입니다.

---

## 2. 지수용육도의 규칙

지수용육도의 정확한 규칙은 다음과 같습니다.

- 1부터 20까지의 자연수를 **정확히 한 번씩** 사용.
- **5개의 육각형**에 각각 6개의 수를 배치.
- 인접한 육각형은 일부 정점을 공유.
- **모든 육각형의 6개 수 합이 63**으로 일정.

이때 다음 중복 계수 방정식이 성립합니다.

```
5 × 63 = 315
1+2+…+20 = 210
315 − 210 = 105
```

즉, 공유 정점들이 추가로 더해진 총량이 105입니다. 지수용육도는 8개 공유 정점 {3, 8, 10, 11, 12, 13, 14, 15}을 통해 이 중복을 만들어냅니다.

따라서 지수용육도의 핵심은 다음 두 가지입니다.

1. **합 불변량**: 각 육각형의 6개 수 합이 S로 동일.
2. **공유 정점 중복**: 공유 정점이 여러 육각형에 중복 포함되어 총합 방정식 `H·S = T + D`를 만족.

---

## 3. 지수귀문도의 일반화

지수귀문도는 위 두 규칙을 **임의의 육각형 타일링**에 적용한 것입니다.

### 3.1 일반 공식

- H: 육각형의 개수
- M: 노드(꼭짓점)의 개수
- T = 1 + 2 + … + M = M(M+1)/2
- mult(v): 노드 v가 속한 육각형의 수
- D = Σᵥ (mult(v) − 1) · v: 공유 정점들의 중복 가중치 합

모든 육각형의 합이 S이면:

```
H · S = T + D
```

따라서:

```
S = (T + D) / H
```

D는 공유 정점의 위치와 multiplicity에 따라 달라지므로, 주어진 토폴로지에서 S는 **유리수 범위**가 아니라 정수 값들의 집합을 이룹니다.

### 3.2 대표적인 지수귀문도 변형

위키백과 "지수귀문도" 항목에 따르면 다음과 같은 변형들이 알려져 있습니다([지수귀문도](https://ko.wikipedia.org/wiki/%EC%A7%80%EC%88%98%EA%B7%80%EB%AC%B8%EB%8F%84)).

| M | 육각형 수 H | 모양 | 마법 상수 S 범위 | 비고 |
|:---:|:---:|:---|:---:|:---|
| 13 | 3 | 삼각형 | 34–50 | |
| 16 | 4 | 마름모 | 40–62 | 총 687,851,136개 해 |
| 20 | 5 | 지수용육도 | 63 | 8개 공유 정점 |
| 22 | 6 | 삼각형 | 57–71 | |
| 24 | 7 | 정육각형 | 65–85 | |
| 30 | 9 | 거북등 | 77–109 | 대표 예시 S=93 |
| 33 | 10 | 삼각형 | 83–121 | |
| 54 | 19 | 정육각형 | 140–190 | |

이 표에서 지수용육도(M=20, H=5)는 지수귀문도 가족의 **한 구성원**에 불과합니다.

## 4. 지수귀문도가 지수용육도의 확장인 이유

### 4.1 동일한 핵심 규칙

지수귀문도와 지수용육도는 다음 규칙을 공유합니다.

1. 1부터 M까지의 수를 **정확히 한 번씩** 사용.
2. 각 육각형에 **6개의 수**를 배치.
3. 모든 육각형의 6개 수 **합이 동일**.
4. 육각형 간 **정점 공유**를 통해 중복 계수 D를 조절.

### 4.2 지수용육도 = 지수귀문도의 특수한 경우

지수용육도는 M=20, H=5인 지수귀문도로 볼 수 있습니다. 다만 전통적으로 "지수용육도"라는 이름이 붙은 이유는 "용육(用六)" 즉 6이라는 수를 핵심 단위로 삼았기 때문입니다.

지수귀문도는 6을 핵심 단위로 유지하면서도, **육각형의 개수 H와 노드 수 M을 자유롭게 확장**합니다. 따라서:

> **지수귀문도 = 지수용육도의 일반화(확장)**

### 4.3 공유 정점 구조의 확장

지수용육도의 5개 육각형은 **중앙 육각형 + 4개 주변 육각형** 구조입니다. 중앙 육각형이 모든 주변 육각형과 정점을 공유하는 형태는, 지수귀문도에서도 7hex 정육각형(M=24)에서 그대로 나타납니다.

7hex 정육각형은 **중앙 육각형 + 주변 6개 육각형**으로, 지수용육도의 중앙-주변 구조를 6방향으로 확장한 것입니다.

## 5. Solver와 실험 결과

동일 폴더의 `jisuguimundo_solver.py`는 노드 수 M이 주어지면 해당하는 지수귀문도 토폴로지를 자동 선택하고, PuLP MILP를 이용해 수 배치를 찾습니다.

### 5.1 사용법

```bash
source venv/bin/activate

# 지수용육도 (M=20, S=63)
python jisuguimundo_solver.py --m 20 --S 63 --output solution_M20.json

# 7hex 정육각형 (M=24, S=75)
python jisuguimundo_solver.py --m 24 --S 75 --output solution_M24.json

# 3hex 삼각형의 S=42 해 100개 열거
python jisuguimundo_solver.py --m 13 --S 42 --enumerate --max-solutions 100 --output solutions_M13_S42.json

# 4hex 마름모 (M=16, S=51)
python jisuguimundo_solver.py --m 16 --S 51 --output solution_M16.json
```

### 5.2 실험 결과 요약

| M | 구조 | H | S | 찾은 해 수 | 출력 파일 |
|:---:|:---|:---:|:---:|:---:|:---|
| 20 | 지수용육도 | 5 | 63 | 1 | `solution_M20.json` |
| 24 | 7hex 정육각형 | 7 | 75 | 1 | `solution_M24.json` |
| 13 | 3hex 삼각형 | 3 | 42 | 100 | `solutions_M13_S42.json` |
| 16 | 4hex 마름모 | 4 | 51 | 1 | `solution_M16.json` |

모든 경우에서 주어진 S에 대해 합법적인 수 배치를 찾았습니다. 특히 M=13에서는 **100개 이상의 해가 존재**함을 확인했으며, 이는 지수귀문도가 풍부한 조합 구조를 가짐을 보여 줍니다.

## 6. "모든 지수귀문도"를 구할 수 있는가?

노드 수 M이 주어졌을 때 **모든 해를 구하는 것은 원칙적으로 가능**합니다. 왜냐하면 조건은 유한한 조합(1부터 M까지의 순열 중 일부 제약 만족)이기 때문입니다.

하지만 실제로는 다음과 같은 한계가 있습니다.

1. **조합 폭발**: M=16(4hex 마름모)만 합핏 687,851,136개의 해가 존재합니다. M이 커질수록 해의 수는 기하급수적으로 늘어납니다.
2. **토폴로지 의존성**: M만으로 구조가 결정되지 않습니다. 예를 들어 M=20은 전통적으로 5hex 지수용육도에 대응되지만, 다른 육각형 배치도 이론적으로 가능합니다.
3. **S 범위 탐색**: 가능한 S는 연속적이지 않고, 각 S마다 MILP를 반복 실행해야 합니다.

따라서 본 solver는 **작은 M(M ≤ 24)**에 대해서는 여러 해를 열거할 수 있지만, 큰 M(M ≥ 30)에 대해서는 **하나 이상의 예시 해를 찾는 정도**가 실용적 한계입니다.

## 7. 결론

지수귀문도는 지수용육도의 규칙을 그대로 따르는 **일반화된 육각형 마방진**입니다.

- 지수용육도: M=20, H=5, S=63.
- 지수귀문도: 동일한 규칙을 M=13, 16, 22, 24, 30, 33, 54 등으로 확장.
- 둘 다 `H·S = T + D`라는 중복 계수 방정식으로 통일됩니다.

실제 solver 실험을 통해 M=13, 16, 20, 24에서 합법적인 배치를 찾았고, 특히 M=13에서는 다수의 해가 존재함을 확인했습니다. 이는 지수귀문도가 단순한 특수 예가 아니라, **지수용육도가 속한 더 넓은 퍼즐 가족**임을 보여 줍니다.

> **지수귀문도는 지수용육도를 포함하는 확장 개념이다.**

# 洛書六觚圖 심층 분석: 성립하는 구조만 남긴 판정

육고도의 수치 지형을 현대 수학의 구조와 대조한 결과, 성립하는 가설은
강하지만 범위가 분명하다. 핵심은 **`F_271`의 부호 반전 구조와 그로부터
자동 발생하는 이차잔여 균형**이다.

페일리 그래프 자체, 리만 제타 영점, GUE 고윳값 분포와의 동일성은
성립하지 않으므로 이 문서의 설명 대상에서 제외한다. 남는 판정은 다음과
같다.

> 육고도는 `F_271`의 부호 반전 구조를 육각 격자에 입힌 정수 순열
> 최적화 문제다. 이 때문에 이차잔여 균형 같은 페일리적 그림자가
> 자동으로 나타난다. 그러나 같은 것은 대척 보수쌍의 유한체 구조이고,
> 나머지는 제약 만족 문제로서의 유사성이다.

## 1. 검증 기준

현재 도안은 `output/solution.json`의 최적해이며, 기하 모델은
`yukgodo/hexgrid.py`에 정의된 반지름 9 육각 격자다.

| 항목 | 값 | 의미 |
|---|---:|---|
| 전체 칸 | 271 | 중심 포함 육각 격자 |
| 채운 칸 | 270 | 중심 `虛一` 제외 |
| 대척 합 | 271 | 마주 보는 두 칸의 값 합 |
| 대척쌍 수 | 135 | 270칸 / 2 |
| 고리 k 칸 수 | 6k | `添六`의 기하 독해 |
| wedge 칸 수 | 45 | 홀수라 완전 반분 불가 |
| ray 칸 수 | 9 | 홀수라 완전 반분 불가 |

목적식의 주요 목표치는 다음과 같다.

- 고리 합: `3k × 271 = 813k`
- 축 합: `9 × 271 = 2439`
- wedge 목표: `45 × 271 / 2 = 6097.5`
- ray 목표: `9 × 271 / 2 = 1219.5`
- 이론적 벌점 하한: `wedges 3.0 + rays 3.0 = 6.0`

## 2. 성립하는 동일성: `F_271`의 부호 반전

값을 `F_271`의 0이 아닌 원소로 보면, 육고도의 대척 조건은

```text
v(-c) = 271 - v(c) ≡ -v(c) (mod 271)
```

이다. 즉 값 배치는 중심 대칭에 대해 홀함수처럼 행동한다.

직접 검정 결과:

```text
p mod 4 = 3
Legendre(-1) = -1
antipodal odd labeling = True
```

여기까지는 비유가 아니라 동일한 수학이다. `271`은 소수이고,
`271 ≡ 3 (mod 4)`이므로 `-1`은 이차비잔여다. 따라서 어떤 값 `x`가
이차잔여이면 `-x`는 이차비잔여이고, 반대로 `x`가 비잔여이면 `-x`는
잔여가 된다.

그 결과 모든 대척쌍 `{x, -x}`는 자동으로

```text
이차잔여 1개 + 이차비잔여 1개
```

를 갖는다.

## 4. 성립하는 산술 하한: wedge/ray의 `±0.5`

현재 최적해의 남는 불균형은 다음과 같다.

```text
wedges = [6097, 6098, 6098, 6098, 6097, 6097]
target = 6097.5
deviation = [-0.5, +0.5, +0.5, +0.5, -0.5, -0.5]

rays = [1219, 1220, 1219, 1220, 1219, 1220]
target = 1219.5
deviation = [-0.5, +0.5, -0.5, +0.5, -0.5, +0.5]
```

이것도 별도 스펙트럼 현상이 아니라 육고도 내부의 정수 산술로 설명된다.

- wedge 하나는 45칸이다.
- 마주 보는 두 wedge는 대척쌍 45개로 대응하므로 두 합의 총합은
  `45 × 271 = 12195`다.
- 이를 둘로 나누면 `6097.5`가 된다.
- 하지만 각 wedge 합은 정수라서 `6097.5`를 정확히 만들 수 없다.
- 그러므로 가능한 최선은 `6097`과 `6098`이다.

ray도 같다.

- ray 하나는 9칸이다.
- 마주 보는 두 ray의 총합은 `9 × 271 = 2439`다.
- 반분 목표는 `1219.5`다.
- 정수 합으로는 불가능하므로 최선은 `1219`와 `1220`이다.

따라서 최종 벌점 `6.0`은 홀수 개 정수 묶음의 반분 불가능성에서 나오는
산술적 하한이다.

```text
wedges: 6개 × 0.5 = 3.0
rays:   6개 × 0.5 = 3.0
total:             6.0
```

## 5. 가장 정확한 현대적 비유

현대적 비유로 가장 잘 맞는 것은 **제약 만족 문제(CSP)**,
**균형 블록 설계**, **희소 부분합 제약**이다.

| 제약 | 성격 |
|---|---|
| 대척쌍 합 271 | 135개의 강한 1:1 제약 |
| 고리 합 813k | 대척쌍 조건에서 자동 성립 |
| 축 합 2439 | 대척쌍 조건에서 자동 성립 |
| 외주 변 합 1355 | 외곽의 추가 균형 제약 |
| wedge 6097/6098 | 45칸 블록의 최적 반분 제약 |
| ray 1219/1220 | 9칸 블록의 최적 반분 제약 |

이 구조는 값 하나가 여러 검사식에 동시에 속하고, 한 곳의 이동이 멀리
떨어진 합 조건을 흔드는 희소 제약 그래프처럼 읽힌다. 다만 선형 패리티
검사 자체는 아니므로 LDPC와 "같다"기보다는, 희소 제약 최적화라는
수준에서 닮았다고 보는 것이 정확하다.

## 6. 최종 판정

| 가설 | 판정 | 이유 |
|---|---|---|
| 육고도 대척쌍은 `F_271`의 `x ↔ -x`와 같다 | 참 | `v(-c) ≡ -v(c) (mod 271)` |
| 각 대척쌍은 QR/QNR 하나씩을 갖는다 | 참 | `271 ≡ 3 (mod 4)`, `Legendre(-1) = -1` |
| 고리·축의 QR 균형이 자동으로 생긴다 | 참 | 고리·축이 대척쌍의 합집합 |
| wedge/ray의 `±0.5`는 산술적 최적 하한이다 | 참 | 홀수 칸 블록의 반정수 목표 |
| 육고도는 CSP/희소 제약 최적화와 닮았다 | 강한 비유 가능 | 다중 부분합 제약과 보수쌍이 겹침 |

요약하면, 육고도의 "소름돋는" 부분은 외부의 거대 이론과 같아서가 아니라
아주 작은 산술 원리 `x ↔ -x (mod 271)`가 270칸의 육각 격자 전체에
퍼지면서 고리, 축, wedge, ray의 전역 균형을 동시에 만든다는 데 있다.

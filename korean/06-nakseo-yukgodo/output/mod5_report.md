## 3. 대척점(점대칭)의 잉여류 작용 — 합 271 ≡ 1 (mod 5)

대척쌍 합이 271이므로 잉여류는 r ↦ (1−r) mod 5 로 작용해야 한다.
실측 불일치 쌍: **0건** (0이어야 함).

| 층 | 대척상 |
|---|---|
| 잉여 1 | 잉여 0 |
| 잉여 2 | 잉여 4 |
| 잉여 3 | 잉여 3 |
| 잉여 4 | 잉여 2 |
| 잉여 0 | 잉여 1 |

대척 짝지음 행렬 A[i][j] = i층 셀의 대척점이 j층에 속하는 칸 수 (대각선 외 0이면 층이 닫혀 있음):

| | 잉여0 | 잉여1 | 잉여2 | 잉여3 | 잉여4 |
|---|---|---|---|---|---|
| 잉여0 | 0 | 54 | 0 | 0 | 0 |
| 잉여1 | 54 | 0 | 0 | 0 | 0 |
| 잉여2 | 0 | 0 | 0 | 0 | 54 |
| 잉여3 | 0 | 0 | 0 | 54 | 0 |
| 잉여4 | 0 | 0 | 54 | 0 | 0 |

해석: A[0][1] = A[1][0] = 54 (0층과 1층이 서로의 대척상),
A[2][4] = A[4][2] = 54 (2층과 4층이 서로의 대척상),
A[3][3] = 54 (3층은 자기 자신의 대척상 — 27개 점대칭 쌍).
이것이 §1의 합동 전부다: 0↔1, 2↔4의 회전180° 합동과 3층의
자기대칭은 대척쌍 합 271의 mod 5 잉여류 작용 r ↦ 1−r 그 자체다.
증명 그림: `mod5_symmetry.png` (색칠 층 vs 상대 층의 대척상 ○ 겹침).

## 8. 파생 정리 — mod N 일반화 (기록)

**정리.** 값 배치에 위치 대합 π(π² = id)이 있고 모든 쌍의 값 합이
상수 S이면, 임의의 법 m에 대해 π는 mod m 잉여류를
**r ↦ (S − r) mod m**으로 작용한다. 따라서 잉여류 층은 π-대칭으로
서로 합동(궤도 길이 2)이거나 자기 합동(고정점: 2r ≡ S (mod m)의 해)이다.

까닭 (4단계):
1. 쌍 조건 v + v′ = S 를 mod m으로 내리면 v′ ≡ S − v — 셀마다 성립.
2. 잉여류 위의 작용은 대합 r ↦ S − r 하나뿐이며, 대합의 궤도는
   길이 2의 쌍 아니면 고정점(2r ≡ S)이다.
3. π가 일대일이고 층 크기가 유한하므로, π(층 r) ⊆ 층 (S−r) 은 곧
   집합 동치다 — 겹침이 근사가 아니라 완전한 이유.
4. 본 도안에서 π는 중심 점대칭 = 회전180°이므로, 층 합동은 격자
   대칭 D6 안에서 실현된다 (§1의 전수조사가 실증).

따름정리 (쌍 합의 패리티):
- S가 홀수이면 자기쌍(고정 셀)은 값이 존재할 수 없다 — 본 도안의
  S = 271(홀수)와 중심 虛一이 이에 정합한다.
- S가 짝수이면 고정 셀의 값은 S/2로 강제된다 — 九子角得 중궁의
  중심 23 = 46/2가 실례다.

교차 도안 검증 (`python3 -m yukgodo.modn_generalization`, 법 2..9 전수):

| 도안 | 쌍 합 S | π (위치 대합) | 결과 |
|---|---|---|---|
| 06 洛書六觚圖 (본 최적해) | 271 | 중심 점대칭 (전역 회전180°) | mod 2..9 전부 정확 |
| 02 九子角得 중궁 | 46 | 3×3 중심대칭 | mod 2..9 전부 정확, 자기쌍 23 = S/2 |
| 07 重卦用八圖 가로진 | 65 | 행 내 좌우 반전 (국소) | mod 2..9 전부 정확 |
| 07 侯策用九圖 | ≈73 (불완전) | formation 위치쌍 | 혼합 시 붕괴(외 성분 12건), 합 73인 16쌍만 추리면 정확 |

侯策用九圖의 사례는 조건의 필요성을 보여준다: 쌍 합이 일정하지 않으면
작용은 r ↦ S−r로 모이지 않고 쌍별 실제 합으로 분해.

의의와 한계: 이 성질은 대척쌍 가설을 만족하는 **모든** 해(시드 42
최적해 포함)가 가지므로 원래 배치의 식별력은 없다. 그러나 mod 5 채색은
오자각득(mod5_residue_diagram.py)·하도사오도(5-컬러링 문서) 등
구수략 분석 전반에서 반복 사용된 기법이며, 이 정리는 그 성분-쌍 검산을
임의의 mod N으로 일반화한다. 더 선명한 판본의 실제 도안이 이 대칭을
갖지 않는다면 대척 보수쌍 가설이 기각된다는 의미에서 반증 도구다.

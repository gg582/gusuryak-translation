# Π(5, q, T): 사오도와 칠자각득을 포괄하는 5진 방향-가중 퍼즐 프레임워크

> 이 글은 전통 도식인 사오도와 칠자각득을 현대 조합론 및 이산수학 언어로 재정의하고, 두 퍼즐이 하나의 파라미터화된 족 `Π(5, q, T)` 내에서 어떻게 배치되는지 보여줍니다. 문화학적이거나 수사적인 해석은 배제하고, 관측 가능한 데이터와 수학적 구조에 집중합니다.

---

## 1. 두 퍼즐의 원본 데이터

### 사오도(四五圖) 5-컬러링 도식

```text
        19  2
        7  14
13  8   5   16  4   17
18  3   11  10  12   9
        15  1
        6  20
```

- 20개의 원형 표식
- 총합 `Σ = 210`
- 5개 색 클래스(물, 불, 나무, 금속, 흙), 각각 4개씩
- 색 분류: `c(v) = L(v) mod 5` (0은 5로 대치)

### 칠자각득(七子各得)

| 클러스터 | 중심 | 주변 슬롯 | 합 |
|---|---|---|---|
| C₁ (상) | 2 | 29, 1, 24, 34, 11, 19 | 120 |
| C₂ (좌) | 3 | 6, 33, 23, 13, 34, 8 | 120 |
| C₃ (중) | 5 | 22, 7, 20, 30, 26, 10 | 120 |
| C₄ (우) | 4 | 15, 28, 9, 18, 32, 14 | 120 |
| C₅ (하) | 1 | 35, 16, 21, 24, 6, 17 | 120 |

- 5개 클러스터, 각 7개 원소(중심 1 + 주변 6)
- 각 클러스터 합 `T = 120`
- 중심 집합 `{1,2,3,4,5}`가 5진 완전 잉여계

## 2. 공통 구조 추출

두 도식을 비교하면 다음 공통 골격이 드러납니다.

1. **5개의 방향**: 사오도의 5색 클래스와 칠자각득의 5개 클러스터가 대응.
2. **5진 잉여 분류**: 두 퍼즐 모두 `mod 5`에 기반한 분할이 핵심.
3. **가중 함수**: 각 원소에 자연수 라벨이 부여됨.
4. **합 불변량**: 사오도는 전역 합 210, 칠자각득은 방향별 합 120.
5. **기하 배치**: 원형 표식의 위치가 데이터의 일부임.

차이점도 분명합니다. 칠자각득은 명시적인 중심-주변 분할과 연결선을 갖지만, 사오도는 변이 없고 중심 개념도 원문에 명시되지 않았습니다.

---

## 3. 일반 프레임워크 `Π(p, q, T)`

두 퍼즐을 포괄하는 정의를 다음과 같이 제시합니다.

```text
Π = (D, M, X, E, K, S, W, Φ)
```

| 기호 | 의미 |
|---|---|
| `D` | 방향 집합, `|D| = p` |
| `M` | 원소/표식 집합 |
| `X` | 기하 배치 `X: M → ℝ²` |
| `E` | 변 집합 `E ⊆ M × M` |
| `K` | 방향-중심 대응 `K: D → M` |
| `S` | 방향별 주변 슬롯 `S: D → 2^M` |
| `W` | 가중 함수 `W: M → ℕ` |
| `Φ` | 불변량 조건 |

이를 세 개의 양의 정수 파라미터로 압축하면:

```text
Π(p, q, T)
```

- `p`: 방향 수
- `q`: 각 방향의 주변 슬롯 수
- `T`: 각 방향의 목표 합

두 퍼즐의 대응은 다음과 같습니다.

| 퍼즐 | p | q | T |
|---|---|---|---|
| 칠자각득 | 5 | 6 | 120 |
| 사오도(색 클래스) | 5 | 3 | 34, 38, 42, 46, 50 |

## 4. 따름 판정

칠자각득의 엄격한 규칙을 사오도에 적용하면 다음과 같습니다.

| 단계 | 조건 | 사오도 만족 여부 |
|---|---|---|
| 0 | `p = 5`인 방향 구조 | 예 |
| 1 | 각 방향의 중심-주변 분할 가능 | 예 (주변 3개) |
| 2 | `q = 6` | 아니오 |
| 3 | 모든 방향의 합이 `T`로 일정 | 아니오 |
| 4 | 명시적 중심-주변 변 존재 | 아니오 |

따라서 사오도는 칠자각득의 **약한 골격**은 공유하지만 **강한 구조**는 따르지 않습니다. 같은 족 `Π(5, q, T)`에 속하는 비균일 파라미터 설정으로 보는 것이 정확합니다.

## 5. 핵심 수학적 제약

### 5.1 5진 잉여 완전성

두 퍼즐 모두 5개 중심이 `ℤ/5ℤ`의 완전 잉여계를 형성합니다.

```text
{1, 2, 3, 4, 5}  →  잉여 {1, 2, 3, 4, 0}
```

### 5.2 합 불변량

- 사오도: `Σ_{v∈M} W(v) = 210 = 1 + 2 + … + 20`
- 칠자각득: `∀d ∈ D, W(K(d)) + Σ_{s∈S(d)} W(s) = 120`

### 5.3 중복 및 배치

- 사오도: 20개 숫자가 모두 서로 다름.
- 칠자각득: 1, 6, 24, 34가 두 번씩 등장(중복). 31개 고유 값.

### 5.4 기하학

- 사오도: 대칭 십자가형, 행 크기 `2, 2, 6, 6, 2, 2`.
- 칠자각득: 5개 별 모양 배치가 십자가로 배치, 각 별은 중심 + 정육각형 주변.

## 6. 파이썬 코드

다음 코드는 두 퍼즐의 핵심 조건을 검증합니다.

```python
#!/usr/bin/env python3
"""
Compare Saodo and Chiljagakdeuk within Π(p, q, T).
"""

# Saodo color classes
SAODO = {
    "water": [1, 6, 11, 16],
    "fire":  [2, 7, 12, 17],
    "wood":  [3, 8, 13, 18],
    "metal": [4, 9, 14, 19],
    "earth": [5, 10, 15, 20],
}

# Chiljagakdeuk clusters
CHILJA = {
    "top":    {"center": 2,  "periphery": [29, 1, 24, 34, 11, 19]},
    "left":   {"center": 3,  "periphery": [6, 33, 23, 13, 34, 8]},
    "center": {"center": 5,  "periphery": [22, 7, 20, 30, 26, 10]},
    "right":  {"center": 4,  "periphery": [15, 28, 9, 18, 32, 14]},
    "bottom": {"center": 1,  "periphery": [35, 16, 21, 24, 6, 17]},
}


def residue(value):
    return ((value - 1) % 5) + 1


def check_chiljagakdeuk():
    centers = [d["center"] for d in CHILJA.values()]
    sums = [d["center"] + sum(d["periphery"]) for d in CHILJA.values()]
    qs = [len(d["periphery"]) for d in CHILJA.values()]
    return {
        "p": len(CHILJA),
        "centers_complete": sorted(residue(c) for c in centers) == [1, 2, 3, 4, 5],
        "q_uniform": all(q == 6 for q in qs),
        "sum_uniform": len(set(sums)) == 1,
        "T": sums[0] if len(set(sums)) == 1 else None,
    }


def check_saodo():
    centers = [min(v) for v in SAODO.values()]
    sums = [sum(v) for v in SAODO.values()]
    qs = [len(v) - 1 for v in SAODO.values()]
    return {
        "p": len(SAODO),
        "centers_complete": sorted(residue(c) for c in centers) == [1, 2, 3, 4, 5],
        "q_uniform": all(q == qs[0] for q in qs),
        "q": qs[0],
        "sum_uniform": len(set(sums)) == 1,
        "sums": sorted(sums),
    }


if __name__ == "__main__":
    print("Chiljagakdeuk:", check_chiljagakdeuk())
    print("Saodo:        ", check_saodo())
```

실행 결과:

```text
Chiljagakdeuk: {'p': 5, 'centers_complete': True, 'q_uniform': True, 'sum_uniform': True, 'T': 120}
Saodo:         {'p': 5, 'centers_complete': True, 'q_uniform': True, 'q': 3, 'sum_uniform': False, 'sums': [34, 38, 42, 46, 50]}
```

## 9. 마무리

사오도와 칠자각득은 겉보기에는 다른 두 도식이지만, `Π(5, q, T)`라는 공통 프레임워크 안에서 같은 족의 다른 인스턴스로 이해할 수 있습니다. 칠자각득은 엄격한 균일 파라미터 `Π(5, 6, 120)`을, 사오도는 비균일 파라미터 `Π(5, 3, T_d)`을 보여줍니다. 이처럼 전통 도식을 현대 조합론 언어로 재정의하면, 단순한 해석을 넘어 명확한 수학적 문제와 일반화 가능성을 얻을 수 있습니다.

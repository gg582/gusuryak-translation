# 각득(各得) 원리란 무엇인가: 구·오·육·칠·팔자각득을 아우르는 구조

《구수략(九數略)》 계열에 전하는 각득(各得) 퍼즐은 "각자 얻는다"는 뜻처럼, 여러 개의 부분 집합이 똑같은 합을 가지도록 수를 배치한 도상입니다. 구자각득(9), 오자각득(5), 육자각득(6), 칠자각득(7), 팔자각득(8) — 이름의 숫자는 각 부분 집합의 크기를 나타냅니다.

## 핵심 질문: 이 퍼즐들은 무엇을 공유하는가?

### 1. 모두 5개의 부분 집합

각득 계열의 첫 번째 공통점은 **부분 집합이 5개**라는 것입니다. 이 5개는 공간적으로 다음과 같은 십자가 형태를 따릅니다.

```
        상(上)
          │
    좌(左) ─ 중(中) ─ 우(右)
          │
        하(下)
```

왜 5개일까요? 5는 오행(수·화·목·금·토)의 수이고, 동서남북 사방에 중앙을 더한 5방위의 수이기도 합니다. 따라서 각득 퍼즐은 단순한 수학 퍼즐을 넘어 **전통 우주론의 공간 모델**을 수로 표현한 것으로 볼 수 있습니다.

### 2. 모든 부분 집합의 합이 같다

각득 퍼즐의 핵심 규칙은 **"각자 得한다"**는 점입니다.

| 계열 | 부분 집합 크기 | 각 부분 집합 합 |
|:---:|:---:|:---:|
| 오자각득 | 5 | 65 |
| 육자각득 | 6 | 63 |
| 칠자각득 | 7 | 120 |
| 팔자각득 | 8 | 164 |
| 구자각득 | 9 | 207 |

이 합은 단순히 정해진 숫자가 아니라, 사용된 수들의 평균에 부분 집합 크기를 곱한 값입니다. 예를 들어 팔자각득은 1–40의 평균 20.5에 8을 곱하면 164가 됩니다.

### 3. 중앙이 코어 역할

모든 계열에서 **중앙 부분 집합**이 전체 구조의 핵심입니다.

- 육자각득: 중앙 육각형의 6개 정점이 모두 공유 정점.
- 팔자각득·구자각득: 중궁이 4방향 궁과 grid로 연결.
- 칠자각득: 중심 클러스터가 4방향과 함께 배치.

중앙을 제거하면 4개의 부분 집합이 떨어져 나가는 구조가 반복됩니다.

### 4. 오행 mod 5 등차수열

수를 5로 나눈 나머지로 수·화·목·금·토로 분류하면, 각 오행 군의 합이 등차수열을 이룹니다.

| 계열 | 오행 합 |
|:---|:---|
| 육자각득 | 34, 38, 42, 46, 50 |
| 팔자각득 | 148, 156, 164, 172, 180 |
| 구자각득 | 189, 198, 207, 216, 225 |

공차는 각 오행 군에 들어간 수의 개수와 같습니다. 이는 1, 2, 3, …을 5개씩 묶었을 때 필연적으로 나타나는 패턴입니다.

### 5. 상극이 우세한 오행 관계

부분 집합 간 연결을 오행 관계로 볼 때, **상극(相剋)** 관계가 가장 많이 등장합니다.

- 육자각득: 상극 50%
- 팔자각득: 상극 55.8%
- 구자각득: 상극 40.3%

이는 각득 퍼즐이 단순한 조화를 넘어 **오행 간 긴장과 제약**을 시각화한 구조임을 보여 줍니다.

## 각 계열의 개성

### 오자각득(천수용오도): 기하 속의 합 불변량

오자각득도 5개 부분 집합의 합이 **65**인 것으로 확인되었습니다. 5×65 = 325이며, 이는 사용된 21개 수의 총합 265보다 60만큼 큽니다. 이 60은 부분 집합들이 공유하는 정점들의 중복 합입니다. 여기에 더해 **좌우 대칭 합 = 86** 같은 위치 기반 불변량도 존재합니다.

### 칠자각득: 중심-주변 분할

칠자각득은 중심 1개 + 주변 6개로 이루어집니다. 중심이 `{1, 2, 3, 4, 5}`로 `mod 5` 완전 잉여계를 이루는 점이 독특합니다.

### 구자각득·팔자각득: 3×3 Grid 가족

둘 다 3×3 격자를 기반으로 합니다. 구자각득은 9칸을 모두 사용하고, 팔자각득은 중심을 비운 8칸을 사용합니다. 모서리/변의 중점 합에서 강력한 위치 불변량이 발견됩니다.

## 일반화: 각득 퍼즐의 공식

각득 퍼즐을 추상화하면 다음과 같이 표현할 수 있습니다.

```
5개 부분 집합, 각각 n개 정점
모든 부분 집합의 합 = S
전체 수의 합 = T
중복으로 더해진 양 = D
5·S = T + D
```

- 부분 집합이 정점을 공유하지 않으면: D = 0
- 부분 집합이 정점을 공유하면: D > 0
- 칠자각득처럼 중복된 숫자 값이 있으면: D는 중복 값들의 합

또한 오행 합의 일반식은 다음과 같습니다.

```
WX(r) = 5·m(m−1)/2 + m·r    (r = 1, 2, 3, 4, 5)
```

여기서 `m`은 각 오행 군에 들어간 수의 개수입니다.

## 별첨: N자각득 Solver

이 폴더에는 `n`만 주면 각득 퍼즐을 자동으로 찾아주는 Python + PuLP MILP Solver가 포함되어 있습니다. SageMath가 없어도 가상환경에서 실행할 수 있습니다.

```bash
source venv/bin/activate
python gakdeuk_solver.py --all --time-limit 180 --max-solutions 1 --visualize --output solutions.json
```

이 solver는 단순한 수 조합을 넘어 다음과 같은 기하·오행 제약을 한꺼번에 다룹니다.

1. **기하적 공유 제약**: 공유 정점은 인접한 클러스터들 사이에서만 발생합니다. `cross`(십자가), `honeycomb`(육각 벌집), `grid`(3×3 격자) 인접 그래프를 기본 제공하며, 사용자가 인접 쌍을 직접 지정할 수도 있습니다.
2. **인접 쌍별 공유 정점 수**: 예를 들어 중앙-상향 쌍이 2개, 중앙-좌측 쌍이 2개씩 정점을 공유하도록 강제할 수 있습니다. 이를 통해 지수용육도나 천수용오도의 세부 기하 구조를 재현할 수 있습니다.
3. **상하/좌우 residue 대칭성**: `--symmetry ud|lr|both` 옵션으로 클러스터의 `mod 5` remainder 분포를 대칭으로 맞춥니다. 전통 도상의 좌우 또는 상하 거울 대칭을 수치적으로 반영합니다.
4. **mod 5 remainder(오행) 동치류 제약**: `--residue-balance`로 각 클러스터의 오행 분포가 전체 분포에 맞춰 균형 있게 배분되도록 유도합니다.

또한 solver는 찾은 해를 **JSON으로 저장**(`--output`)하고, **matplotlib 시각화**(`--visualize`)까지 한 번에 생성합니다. `--max-solutions`로 여러 합법적 배치를 열거할 수도 있습니다.

```bash
# 육자각득 유사: 벌집 인접 + 쌍별 공유 정점 수 + 상하 대칭
python gakdeuk_solver.py --n 6 --max 20 --sum 63     --adjacency honeycomb --pair-shared-counts "0-1:2,0-2:2,0-3:2,0-4:2"     --symmetry ud --residue-balance --visualize

# 모든 알려진 계열을 한 번에 탐색·저장·시각화
python gakdeuk_solver.py --all --time-limit 180     --max-solutions 1 --visualize --output solutions.json
```

이렇게 하면 오자각득·육자각득·칠자각득·팔자각득·구자각득 모두에서 각득 원리를 따르는 해를 찾고, `solutions.json`과 `viz_*.png` 파일로 기록할 수 있습니다.

## 결론

각득 계열은 서로 다른 크기와 형태를 가지지만, 다음 5가지 원리를 공유합니다.

1. **5개 부분 집합 + 중앙 코어**
2. **합 불변량**
3. **중복 계수 방정식 `5·S = T + D`**
4. **오행 mod 5 등차수열**
5. **상극 우세한 오행 관계**

이 원리들은 전통 수리학과 현대 조합론·그래프 이론이 만나는 지점을 보여 주며, 각득 퍼즐을 단순한 암산 놀이가 아닌 **수와 공간의 조화를 탐구하는 도상**으로 재발견하게 합니다.

# 후책용구도

리브레위키의 후책용구도 서술에 따르면 이 도상은 1..72의 72개 원소를
사용하고, 팔각형 13개와 사각형 12개가 함께 배열된다. 팔각형 여덟
수의 합은 292, 사각형 네 수의 합은 146이다.

핵심 유도법은 다음처럼 정리할 수 있다.

- 팔각형의 사선 변을 이루는 두 수가 합 73을 만들도록 한다.
- 그러면 팔각형 하나의 합은 `73 × 4 = 292`로 유도된다.
- 사각형은 한쪽 두 수의 합이 74, 반대쪽 두 수의 합이 72가 되도록
  배치하면 `74 + 72 = 146`을 만족한다.
- 이 배치는 1..36을 한쪽 기준 영역에 놓고, 그 보수 방향으로 37..72를
  대응시키는 방식으로 읽을 수 있다.

현재 폴더의 `visualize.py`는 전체 13팔각형/12사각형 테셀레이션을
완전히 모델링한 파일이라기보다, 3×3 작업 단위의 네 변 쌍을 시각화한
파일이다. 따라서 규칙 검증은 `analyze_huchaek_yonggudo.py`에서 실제
쌍합과 단위합을 따로 계산한다.

원본 재구본 역시 리브레위키의 후책용구도 규칙을 따르는 도상으로 본다.
다만 이 판본은 1..72의 완전 사용과 모든 단위합을 숫자적으로 최적화한
판본이라기보다, 원문 도상의 형태와 국소 쌍합 표기를 우선 보존한
보수적인 재구성이다. 따라서 원본 재구본의 일부 수치 이탈은 규칙
부정이 아니라, 도상 중심 재구성과 숫자 완전 정정본 사이의 차이로
해석한다.

정정본은 `huchaek_data.py`의 `CORRECTED_FORMATIONS`에 분리해 두었다.
정정본에서는 1..72가 정확히 한 번씩 사용되고, 변 쌍 합 분포가
`73` 18개, `74` 9개, `72` 9개가 되어 모든 작업 팔각형의 합이 292가
된다. 자세한 내용은 `corrected_analysis.md`를 참고한다.

## 파일

- `huchaek_data.py` — 작업 데이터
- `visualize.py` — 원본 재구본 3×3 작업 단위 시각화
- `analyze_huchaek_yonggudo.py` — 후책용구도 규칙 검증
- `visualize_huchaek_rule_pairs.py` — `73`, `74`, `72` 쌍합 구조 시각화
- `visualize_corrected.py` — 정정본 시각화
- `visualize_corrected_rule_pairs.py` — 정정본 쌍합 구조 시각화
- `corrected_analysis.md` — 정정본 연구

## 원문

六十四子總積一千
八十以八子為一隊
各得一百六十數
變則八隊化為二十
四隊再變則為一百
五十六隊演積六萬
六千五百六十合於
武侯陣圖

## 설명
- 64개의 수를 사용하며, 원문에는 총적이 1000이라고 적혀 있다.
- 8개씩 묶어 하나의 '대(隊)'를 만든다.
- 각 대의 합은 160이 된다.
- 이를 변형하면 8개의 대는 20개의 형태로 변한다.
- 다시 변형하면 100개의 형태가 된다.
- 모두 합하면 56개의 진형(隊)이 만들어지며,
  전체 경우는 6,560과 대응한다.
- 이것은 무후진도(武侯陣圖), 즉 제갈량의 팔진도와 관련된다.

## 데이터 정정

원문 전사본에는 전사 오류가 섞여 있다. 자세한 탐구 과정은
`transcription_analysis.md`를 참고한다.

### 발견된 문제
- 8개 진형 중 4개의 합이 260이 아니다.
- **14**와 **22**가 중복되어 등장하고, **20**과 **29**가 누락되었다.
- 8개 진형의 총합이 1..64의 총합 2080보다 13 작다.

### 채택한 정정안

원문 위상을 최대한 유지하면서 모든 진형의 합을 260으로 복원한다.

| 위치 | 변경 내용 |
|---|---|
| 1행 1열 (세로) | 14 → 20, 25 → 29 |
| 1행 2열 (가로) | 33 → 35 |
| 1행 3열 (세로) | 22 → 25 |
| 2행 1열 (세로) | 35 → 33 |

- 33 ↔ 35 교환은 1행 2열과 2행 1열 사이의 의도된 변진(±2 대응) 관계를 보존한다.
- 14 → 20, 25 → 29, 22 → 25는 중복·누락을 해소한다.

### 생성 파일

- `visualize_wrong_origin.py` — 원문 전사본(오류 포함) 시각화 (`eight_formation_diagram_wrong_origin.png`, `.svg`)
- `visualize_corrected.py` — 정정본 시각화 (`eight_formation_diagram_corrected.png`, `.svg`)

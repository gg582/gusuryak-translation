# 08. 위치 대칭 보수쌍 도안의 mod N 잉여류 일반화

구수략 도안들에 공통으로 나타나는 위치 대칭 보수쌍(자리 쌍의 값 합이 일정)
구조가 임의의 법에 대해 만들어 내는 잉여류 대칭에 대한 교차 장(章)
분석. 검증 코드는 `korean/06-nakseo-yukgodo/` 프로젝트에 있다.

## 배경: mod 5 채색의 전통

mod 5 잉여류 채색은 이 저장소 전반에서 반복적으로 사용된 분석 기법이다.

- `01-saodo-family/하도사오도-지만-사실-5-컬러링-문제/` — 하도사오도가
  사실상 5-컬러링 문제라는 분석
- `02-gakdeuk-series/오자각득(천수용오도)/mod5_residue_diagram.py` —
  값의 mod 5 잉여류 그룹 도식화
- `06-nakseo-yukgodo/yukgodo/mod5.py` — 복원 최적해를 mod 5 잉여류별
  5개 층(각 54칸)으로 분리하고, 층 간 D6 대칭 12원소 × 전 층쌍을 전수조사

06 장의 전수조사에서 발견된 사실: 잉여 2↔4, 1↔0 층이 회전180°(점대칭)로
54/54 완전 합동이고, 잉여 3 층은 자기대칭이며, 그 외의 대칭은 없다
(다른 쌍의 최대 겹침 13~17/54 = 무작위 수준). 이 장은 이 발견이 mod 5와
육고도에 특수한 것이 아니라, 위치 대칭 보수쌍을 가진 모든 도안으로
일반화됨을 정리하고 검증한다.

## 파생 정리

**정리.** 값 배치에 위치 대합 π(π² = id)가 있고 모든 쌍의 값 합이 상수 S
이면, 임의의 법 m에 대해 π는 mod m 잉여류를

```
r ↦ (S − r) mod m
```

으로 작용한다. 따라서 잉여류 층은 π-대칭으로 서로 합동(궤도 길이 2)이거나
자기 합동(고정점: 2r ≡ S (mod m)의 해)이다.

**까닭 (4단계).**

1. 쌍 조건 v + v′ = S를 mod m으로 내리면 셀마다 v′ ≡ S − v.
2. 잉여류 위의 작용은 대합 r ↦ S − r 하나뿐이며, 대합의 궤도는 길이 2의
   쌍 아니면 고정점(2r ≡ S)이다.
3. π가 일대일이고 층 크기가 유한하므로 π(층 r) ⊆ 층 (S−r)은 곧 집합
   동치다 — 겹침이 근사가 아니라 완전한 이유.
4. π가 격자의 기하 대칭이면(예: 육고도의 중심 점대칭 = 회전180°) 층
   합동이 격자 대칭군 안에서 실현된다.

**따름정리 (쌍 합의 패리티).**

- S가 홀수이면 자기쌍 고정 셀의 값이 존재할 수 없다 — 고정 셀은 비어
  있어야 한다. 육고도의 S = 271(홀수)과 중심 虛一이 이에 정합한다.
- S가 짝수이면 고정 셀의 값은 S/2로 강제된다. 구자각득 중궁의 중심
  23 = 46/2가 실례다.

## 교차 도안 검증

`python3 -m yukgodo.modn_generalization` (06 프로젝트), 법 2..9 전수.
대척 짝지음 행렬 A[i][j] (값 ≡ i인 셀의 대척점 값이 ≡ j인 쌍의 수)의 비영
성분이 전부 (i, S−i) 궤도뿐인지 검사한다.

| 도안 | 쌍 합 S | 위치 대합 π | 결과 |
|---|---|---|---|
| 06 洛書六觚圖 (복원 최적해) | 271 | 중심 점대칭 (전역 회전180°) | mod 2..9 전부 정확 |
| 02 九子角得 중궁 | 46 | 3×3 중심대칭 | 전부 정확, 자기쌍 23 = S/2 |

| 07 侯策用九圖 | ≈73 (불완전) | formation 위치쌍 | 혼합 시 붕괴(외 성분 12건), 합 73인 16쌍만 추리면 정확 |

侯策用九圖는 조건의 필요성을 보여준다: 쌍 합이 일정하지 않으면(73×16쌍,
74×9쌍, 72×7쌍, 이탈 4쌍) 잉여류 작용이 r ↦ S−r로 모이지 않고 쌍별
실제 합으로 갈라진다. 합 73인 쌍만 추리면 패턴이 다시 정확해진다.

### mod 5에서의 궤도 구조 (S별)

| S mod 5 | 궤도 | 고정점 |
|---|---|---|
| 271 ≡ 1 (육고도) | {0,1} {2,4} | {3} |
| 46 ≡ 1 (구자각득 중궁) | {0,1} {2,4} | {3} |
| 65 ≡ 0 (팔진도 가로진) | {1,4} {2,3} | {0} |

육고도와 구자각득 중궁은 S ≡ 1 (mod 5)로 같은 궤도 구조를 가지므로,
mod 5 채색에서 같은 쌍대(0↔1, 2↔4, 3 자기)가 나타난다.

## 의의와 한계

1. **성분-쌍 검산의 일반화.** 각 장에서 mod 5 채색 성분으로 하던 보수쌍
   확인이, 이 정리에 의해 임의의 mod N에서의 동일한 절차로 정당화된다.
2. **반증 도구.** 위치 대칭 보수쌍을 주장하는 복원안은 해당 궤도 구조를
   반드시 가져야 한다. 실제 판본이 이를 어기면 보수쌍 가설이 기각된다.
3. **식별력 없음 (한계).** 이 성질은 상수 쌍 합 조건을 만족하는 모든
   배치가 가지므로(육고도의 서로 다른 최적해들조차 모두), 이것만으로
   원래 배치를 특정할 수는 없다.

# 낙서구구도 분석

낙서구구도는 낙서 정규마방진의 3x3 중심 배열에, 각 중심수마다 여덟 개의 외곽수를 붙여 만든 1~81 마방진이다. 각 묶음은 중심수 하나와 주변 여덟 수로 이루어지며, 아홉 묶음의 합은 모두 369가 된다.

```text
8 1 6
3 5 7
4 9 2
```

## 전체 합과 목표 상수

1부터 81까지의 합은 다음과 같다.

```text
1 + 2 + ... + 81 = 81 x 82 / 2 = 3321
```

이를 아홉 묶음으로 나누면 한 묶음의 목표 합은 다음과 같다.

```text
3321 / 9 = 369
```

따라서 중심수가 `n`인 묶음의 주변 여덟 수는 반드시 다음 합을 가져야 한다.

```text
주변합 = 369 - n
```

중심수가 클수록 주변합은 작아지고, 중심수가 작을수록 주변합은 커진다.

## 91 보수쌍

외곽에 쓰이는 수는 10~81이다. 이 72개 수는 합이 91인 36개의 보수쌍으로 나눌 수 있다.

```text
10 + 81 = 91
11 + 80 = 91
12 + 79 = 91
...
45 + 46 = 91
```

만약 각 중심 주변에 91 보수쌍 네 개를 그대로 배치하면, 주변 여덟 수의 합은 항상 다음과 같다.

```text
91 x 4 = 364
```

그러면 중심수까지 더한 합은 `364 + n`이 되므로, 중심수가 5인 묶음만 369가 된다. 낙서구구도는 이 불균형을 해결하기 위해 네 쌍 중 세 쌍은 91 보수쌍으로 두고, 나머지 한 쌍만 중심수에 따라 조정한다.

## 중심수 보정쌍

세 개의 91 보수쌍을 먼저 쓰면 그 합은 273이다.

```text
91 x 3 = 273
```

중심수 `n`과 나머지 한 쌍의 합을 더해 369를 만들려면, 나머지 한 쌍의 합은 다음이어야 한다.

```text
보정쌍 합 = 369 - n - 273 = 96 - n
```

따라서 중심수별 보정쌍의 합은 87부터 95까지 정확히 한 번씩 나타난다.

| 중심수 | 보정쌍 합 |
|---:|---:|
| 9 | 87 |
| 8 | 88 |
| 7 | 89 |
| 6 | 90 |
| 5 | 91 |
| 4 | 92 |
| 3 | 93 |
| 2 | 94 |
| 1 | 95 |

이 표가 낙서구구도의 핵심이다. 중심수의 차이를 외곽 보정쌍의 차이로 상쇄하기 때문에, 모든 묶음의 합이 369로 같아진다.

## 현재 도상의 검산

현재 `visualize.py`에 들어 있는 낙서구구도 배치는 다음과 같이 검산된다.

| 중심 | 보정쌍 | 보정쌍 합 | 주변합 | 전체합 |
|---:|---:|---:|---:|---:|
| 8 | 46 + 42 | 88 | 361 | 369 |
| 1 | 54 + 41 | 95 | 368 | 369 |
| 6 | 47 + 43 | 90 | 363 | 369 |
| 3 | 53 + 40 | 93 | 366 | 369 |
| 5 | 52 + 39 | 91 | 364 | 369 |
| 7 | 51 + 38 | 89 | 362 | 369 |
| 4 | 48 + 44 | 92 | 365 | 369 |
| 9 | 50 + 37 | 87 | 360 | 369 |
| 2 | 49 + 45 | 94 | 367 | 369 |

각 묶음에는 보정쌍 하나와 91 보수쌍 세 개가 들어 있다. 예를 들어 중심수 8의 경우는 다음과 같다.

```text
17 + 74 = 91
35 + 56 = 91
26 + 65 = 91
46 + 42 = 88

8 + 91 + 91 + 91 + 88 = 369
```

중심수 1의 경우는 다음과 같다.

```text
10 + 81 = 91
28 + 63 = 91
19 + 72 = 91
54 + 41 = 95

1 + 91 + 91 + 91 + 95 = 369
```

## 생성 공식

현재 배치는 중심수 `n`에 대해 세 개의 91 보수쌍을 다음 꼴로 만든다.

```text
(n + 9)  + (82 - n) = 91
(n + 27) + (64 - n) = 91
(n + 18) + (73 - n) = 91
```

나머지 한 쌍은 `96 - n`이 되도록 배치한다. 이 보정쌍들이 중심수 9부터 1까지 87, 88, ..., 95를 하나씩 담당한다.

## 해석

낙서구구도는 단순히 팔각형 외곽합을 같게 만든 도상이 아니다. 중심에 놓인 낙서마방진의 수가 서로 다르기 때문에, 외곽의 한 쌍이 그 차이를 보정하도록 설계되어 있다.

요약하면 다음 구조이다.

```text
각 중심 주변 = 91 보수쌍 3개 + 중심수 보정쌍 1개
보정쌍 합 = 96 - 중심수
중심수 + 주변 여덟 수 = 369
```

따라서 낙서구구도는 1~81 전체를 아홉 개의 369 묶음으로 나누는 확장 낙서마방진이며, 그 생성 원리는 91 보수쌍과 87~95 보정쌍의 결합으로 설명할 수 있다.

# 하도 퍼즐에서 추출한 현대 조합론 문제

> 산학(算學)적 도식에서 현대 조합론·이산수학 언어로 추출한 문제들입니다.

---

## 문제 1. 대칭 십자가형 위의 적절한 5-컬러링

20개의 원형 슬롯이 대칭 십자가형으로 배치되어 있다. 각 슬롯에 1부터 20까지의 자연수가 하나씩 부여되었을 때, 색칠 함수

```
c(v) = L(v) mod 5   (단, 0은 5로 대치)
```

를 정의하라. 이 함수가 5개의 색 클래스로 정확히 4개씩 분할하는 **적절한 5-컬러링**임을 증명하라.


원문 “河圖五五卽上天數圖，六五卽上地數圖”에 따라 1부터 10까지의 수를

- 하늘의 수 `H = {1,3,5,7,9}`
- 땅의 수 `E = {2,4,6,8,10}`

로 분할하라. 다음 항등식을 증명하라.

```
ΣH = 25 = 5²
ΣE = 30 = 5·6
ΣH + ΣE = 55
```

이를 1부터 20까지 확장한 `H' = H ∪ (H+10)`, `E' = E ∪ (E+10)`에 대해서도 `|H'| = |E'| = 10` 및 `ΣH' = 100`, `ΣE' = 110`임을 보여라.


5개 오행 집합 `G = {물, 불, 나무, 금속, 흙}` 위에 다음 두 관계를 정의하라.

- 상극 `σ`: 물 ↔ 흙
- 상보 `τ`: 불 ↔ 금속

`σ`와 `τ`가 모두 `G` 위의 대합임을 증명하고, `σ² = τ² = Id`임을 확인하라. 또한 `σ ∘ τ = τ ∘ σ`인지 조사하라.


`V = {1,…,20}` 위에 두 개의 블록족을 정의하라.

```
B_H = { {1,6,11,16}, {2,7,12,17}, {3,8,13,18}, {4,9,14,19}, {5,10,15,20} }
B_E = { {1,11}, {3,13}, {5,15}, {7,17}, {9,19} }
```

`B_H`와 `B_E`가 각각 `V`의 부분집합족임을 확인하고, 두 블록족 사이의 교차 행렬

```
M[i,j] = |B_H[i] ∩ B_E[j]|
```

을 계산하라. 이 행렬이 어떤 구조적 특징을 가지는지 설명하라.


오행 집합 `G` 위에 다음 재작성 규칙을 정의하라.

```
나무 + 나무 → 불
금속 + 불 → ∅   (소거)
```

이 규칙을 현대 항 재작성 시스템의 일부로 볼 때, 다음 질문에 답하라.

1. 이 규칙들이 **합류성**을 갖는가?
2. 주어진 초기 다중집합 `{나무, 나무, 금속, 불}`에서 유도할 수 있는 정상형은 무엇인가?
3. 검산 `Σ L(v) = 210`이 재작성 과정에서 어떻게 변하는가?


도식의 모든 원형 슬롯에 적힌 숫자의 합이 `共積210`이라는 조건이 주어졌다. 이것이 1부터 20까지의 합임을 증명하고, 만약 “나무+나무→불” 또는 “금속+불→소거”와 같은 재작성 규칙이 실제로 적용된다면, 어떤 추가 불변량을 정의해야 총합이 의미를 잃지 않는지 논하라.

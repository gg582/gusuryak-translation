# 사오도(四/五道)와 각득(各得) 계열의 상호 해석

낙서사구도(洛書四九圖)와 낙서오구도(洛書五九圖)는 원문이 각각 **사자각득(四子各得)**과 **오자각득(五子各得)**으로 직접 읽히지만, 동시에 **사오도(四/五道)** 해석도 가능하다.  
아래에서는 같은 도상을 두 체계에서 **상호 변환**하여 읽고, unification 관점에서 공통점과 차이를 정리한다.

---

## 1. 원문과 기본 수치

### 낙서사구도 → 사자각득

> 鄰星相兼五宫化 爲九宫每宫四子 各得四十二數  
> 此即河圖四五画 右旋者 互化則一千八百九十數

|---|---:|
| 사용 수 | 1~20 (20개) |

| 9궁 중복 포함 총합 | 378 |

### 낙서오구도 → 오자각득

> 五子各得 八十五數  
> 九宮共得 七百六十五數

| 항목 | 값 |

## 2. 상호 변환 테이블

### 2.1 각득 → 사오도

| 도상 | 각득 해석 | 사오도 해석 |
|---|---|---|
| 낙서사구도 | 9궁 각각 4자를 얻어 42를 만든다 (사자각득) | 5개 오행 분류를 인접 겹침으로 9블록으로 재구성. 4도(4자)의 부분합을 5도(오행) 위에서 실현. |
| 낙서오구도 | 9궁 각각 5자를 얻어 85를 만든다 (오자각득) | 3×3 십자형 5정점 궁 구조. 5도(5자)의 부분합을 4도(3×3 격자)에 전개. |

### 2.2 사오도 → 각득

| 사오도 구성요소 | 낙서사구도에서의 각득 의미 | 낙서오구도에서의 각득 의미 |
|---|---|---|
| 4도(사도) | 9궁 각각 4자; 중앙 4-cycle | 9궁을 4모서리 + 4변 + 1중앙으로 분할 |
| 5도(오도) | 20개 수를 5개 mod 5 클래스로 분류 | 각 궁의 십자형 5정점 = 중심 1 + 사방 4 |
| 9궁(九宫) | 9개 블록 모두 합 42 | 9개 십자 궁 모두 합 85 |
| 作/用 분리 | 20작 / 36용 | 33작 / 45용 |

## 3. Unification: 공통점과 해석 차이

### 공통점

| 항목 | 사자각득(낙서사구도) | 오자각득(낙서오구도) |
|---|---|---|
| 궁의 수 | 9개 | 9개 |
| 오행 분류 | 5개 residue class | 5개 residue class |
| 부분합 불변량 | 모든 궁의 합 = 42 | 모든 궁의 합 = 85 |
| 중복 계수 합 | 42 × 9 = 378 | 85 × 9 = 765 |
| 핵심 공간 구조 | 중앙 코어 + 4방향 확장 | 중앙 코어 + 4방향 확장 |
| 作/용 구분 | O | O |

### 해석 차이

| 항목 | 사자각득(낙서사구도) | 오자각득(낙서오구도) |
|---|---|---|
| 각 궁의 크기 | 4자 | 5자 |
| 사용 수의 범위 | 1–20 (20개) | 1–33 (33개) |
| 중앙 코어 | 4-cycle (5, 16, 10, 11) | 십자형 5정점 (5, 19, 26, 18, 17) |
| 기하 구조 | 3×2 사각형, 4개 육각 면 | 3×3 격자, 9개 십자형 궁 |
| 원문 합 공식 | 5궁 × 9궁 × 42 = 1,890 | 9궁 × 85 = 765 |
| 강조되는 도(道) | 4도(사도)가 중심 | 5도(오도)가 중심 |
| 현대적 이름 | Sajagakdeuk | Ojagakdeuk |

**요약:** 사자각득은 **4도의 부분합 불변량**을 **5도의 분류 체계** 위에서 실현한 구조이고, 오자각득은 **5도의 부분합 불변량**을 **4도의 3×3 격자**에 전개한 구조다.  
둘은 “각 궁이 동일한 부분합을 갖는다”는 **각득 원리**를 공유하지만, 4도냐 5도냐에 따라 기하와 합의 크기가 달라진다.

## 4. Π(p, q, T) 프레임워크에서의 위치

`04-unification`의 사오도-칠자각득 일반화에서 제시된 파라미터족

```
Π(p, q, T)
```

- `p`: 방향(또는 오행) 수
- `q`: 각 방향의 주변 슬롯 수
- `T`: 각 방향(큰러스터)의 목표 합

| 퍼즐 | p | q | T |
|---|---|---|---|
| 사자각득(9궁, 4자) | 5(오행) | 3(2주기층) | 34, 38, 42, 46, 50 |
| 오자각득(9궁, 5자) | 5(오행) | 6 또는 7 | 99, 105, 112, 119, 126 |

즉 두 도상은 각득 원리라는 큰 틀 아래서 **4와 5의 비율만 다른 두 변주곡**이다.

## 5. 1700년 근세 출판 시기 기준으로의 선진성

- **사오도(四/五道)** 해석은 전통적인 우주론적 틀(방위, 오행, 9궁)을 그대로 사용한다.
- **각득(各得)** 해석은 같은 틀 안에서 **부분합 불변량**이라는 명시적 조합 조건을 드러낸다.
- 따라서 **각득 해석이 더 선진적**이다: 단순히 “마방진처럼 보인다”가 아니라, “모든 부분집합의 합이 같다”는 정량적 규칙을 기술한다.
- 현대적으로 재해석할 때는 다음을 본다:
  1. **정수 선형 방정식** (`Σ_{v∈C} x_v = S` for every cluster C)
  2. **중복 계수 방정식** (`k·S = T + D`)
  3. **mod / CRT 분류** (mod 2, 3, 4, 5, 6, 9, 12)
  4. **cluster 회전 대칭** (rotation_analysis.py)

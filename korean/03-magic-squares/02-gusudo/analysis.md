# 구수도(九數圖)

차수: 9×9

## 정상 마방진 여부

구수도는 **정상 마방진(normal magic square)이다**. 1부터 81까지의 자연수를 한 번씩 사용하며, 모든 행·열·두 대각선의 합이 마방진 상수 369로 일치한다. 교정이 필요 없는 원본 그대로의 정상 마방진이다.

## 생성 규칙 요약

마방진 상수는

\[
M_9 = \frac{9(9^2+1)}{2} = 369
\]

이며, 두 예시 모두 모든 행, 열, 두 대각선의 합이 369이다.

9×9는 홀수 차수이므로 전통적인 Siamese method(연속 배치법)의 변형이나, 최석정이 사용한 것처럼 **직교 라틴 방진(orthogonal Latin squares)**을 결합하는 방식으로 생성할 수 있다. 실제로 구수도의 구조는 9차 직교 라틴 방진을 이용한 마방진 작성법과 밀접하게 관련되어 있다.

### Associated 성질

예시 1에서 중심 대칭 위치의 두 수 합이 모두 82이다. 이는 정상 9×9 마방진에서 associated magic square의 정의와 일치한다.

\[
a_{i,j} + a_{8-i,8-j} = 82
\]

### 예시 2를 associated로 교정하려면

예시 2를 associated magic square로 만들려면, **모든 중심 대칭 위치의 두 수 합이 82**가 되도록 수들을 재배치해야 한다. 즉, 각 수 \(x\)가 채워진 위치의 중심 대칭 위치에는 \(82 - x\)가 있어야 한다.

예시 2에서 중심값 41은 이미 정중앙 \(4,4\)에 위치해 있어 associated의 필요조건은 만족하지만, 나머지 40개의 complementary pair가 중심대칭 쌍으로 배치되지 않았다. 실제로 예시 2의 중심대칭 합은 26부터 138까지 다양하게 흩어져 있다. 예를 들어:

```
(0,0)=50 <-> (8,8)=40: 합=90  (82가 아님)
(0,1)=18 <-> (8,7)= 8: 합=26  (82가 아님)
(0,2)=55 <-> (8,6)=75: 합=130 (82가 아님)
```

따라서 예시 2를 associated로 바꾸는 것은 몇 개의 수만 교환하는 것이 아니라, **40개의 complementary pair를 모두 중심대칭 쌍으로 재배치하는 광범위한 변형**이 필요하다. 이는 본질적으로 새로운 9×9 associated 마방진을 설계하는 것과 같다.

associated 9×9 마방진의 한 예시는 [`corrected_associated.md`](corrected_associated.md)와 시각화 파일 `구수도_associated_교정본.png`에 수록하였다. 이 교정본은 모든 행·열·두 대각선의 합이 369이고, 모든 중심대칭 쌍의 합이 82이다.

### 두 예시의 관계

예시 2도 정상 9×9 마방진이지만 예시 1과는 associated 성질이 다르다. 예시 1은 associated이고 예시 2는 그렇지 않다.

두 예시가 서로 어떤 기본 변환(행 치환, 열 치환, 반전, 회전)으로 연결되는지 확인하면, 예시 2가 예시 1의 어떤 변형인지 더 명확히 알 수 있다. 눈으로 보기에 두 예시는 서로 다른 배치 패턴을 보이며, 단순 반전이나 회전만으로는 얻어지지 않는다.

## 3×3 블록 구조

9×9 마방진을 3×3 블록 9개로 나누면, 각 블록의 합이 일정하지는 않지만, 마방진 상수 369와 관련된 구조를 가진다. 홀수 차수 마방진에서 3×3 블록은 짝수 차수와는 다른 성질을 보인다.

## 역사적 맥락

최석정(1646~1715)은 구수략에서 9×9 마방진을 직교 라틴 방진을 이용해 구성했다. 이는 레온하르트 오일러보다 약 60년 앞선 시도로 평가받는다. 구수도의 구조는 직교 라틴 방진의 합(또는 결합)으로 1~81을 생성한 것으로 보인다.

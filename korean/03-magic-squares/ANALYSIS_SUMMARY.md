# 방진 분석 요약

`square.md`에 수록된 6종류의 방진을 차수별로 폴다에 나누어 분석하였다.

## 폴다 구조

```
01-yukyukdo/                    육육도
02-gusudo/                      구수도
03-baekjajasuyin-yang-chakjong/ 백자자수음양착종도
04-baekjasaengseong-sunsu/      백자생성순수도
05-baekjasaengseong-gyosu/      백자생성교수도
06-baekjayin-yang-jamo-chakjong/ 백자음양자모착종도
```

각 폴다에는 `analysis.md`가 있고, 10×10 방진 폴다에는 추가로 `corrected.md`(정상 마방진 교정본)가 있다.

## 정상 마방진 여부 요약

| 방진 | 차수 | 정상 마방진? | 비고 |
|---|---|---|---|
| **육육도(六六圖)** | 6×6 | **예** | associated는 불가능 (6 ≡ 2 mod 4) |
| **구수도(九數圖)** | 9×9 | **예** | 예시 1은 associated. 예시 2는 `corrected_associated.md`로 교정 가능 |
| **백자자수음양착종도** | 10×10 | **아니오** | 두 라틴 방진의 보완적 쌍. `corrected.md` 참조 |
| **백자생성순수도** | 10×10 | **아니오** | 직교라틴 방진 아님. `corrected.md` 참조 |
| **백자생성교수도** | 10×10 | **아니오** | 순수도와 쌍을 이룸. `corrected.md` 참조 |
| **백자음양자모착종도** | 10×10 | **아니오** | 1~100은 한 번씩, 8/10 행·열이 505. `corrected.md` 참조 |

## 시각화 파일

`figures/` 폴다에 다음 PNG 파일이 생성되었다.

- 육육도: 원본 예시 1, 예시 2 (`육육도_예시1.png`, `육육도_예시2.png`)
- 구수도: 원본 예시 1, 예시 2 (`구수도_예시1.png`, `구수도_예시2.png`) 및 associated 교정본 (`구수도_associated_교정본.png`)
- 10×10 방진: 정상 마방진 교정본 4개 (`백자..._교정본.png`)

## 핵심 수학적 성질

### 마방진 상수

- 6×6: \(M_6 = 6(36+1)/2 = 111\)
- 9×9: \(M_9 = 9(81+1)/2 = 369\)
- 10×10: \(M_{10} = 10(100+1)/2 = 505\)

### Associated Magic Square

중심 대칭 위치의 두 수 합이 \(n^2+1\)로 일정하면 associated magic square이다.
- 구수도 예시 1: \(a_{i,j}+a_{8-i,8-j}=82\)로 완전한 associated.
- 육육도 예시 2: 36쌍 중 32쌍이 37. 6×6 associated magic square는 존재하지 않으므로 완전한 associated가 될 수 없다.

### Pan-diagonal

래핑(wrap)된 모든 대각선의 합도 마방진 상수와 같으면 pan-diagonal이다. 본 문서의 모든 방진은 pan-diagonal이 **아니다**.

### 10×10 교정본

10×10 방진 4종류에 대해 `magic_square` Python 패키지의 `magic(10)`으로 생성한 정상 10×10 마방진을 교정본으로 제시하였다. 각 교정본은 1~100을 한 번씩 사용하고 모든 행·열·두 대각선의 합이 505이다.

## 육육도의 특이점

6×6은 짝수 차수 중에서도 **singly even**(\(n \equiv 2 \pmod 4\))에 속하여 마방진 생성이 복잡하다. 육육도의 두 예시는 모두 정상 마방진이지만:

- 예시 1: associated 성질이 전혀 없음.
- 예시 2: 36개 중심대칭 쌍 중 **32개가 37**(=6²+1)을 만족. 나머지 4개는 2×2 영역에서 깨짐.

그러나 6×6 associated magic square는 존재하지 않으므로, 예시 2도 "거의 associated"일 뿐 완전한 associated로 교정할 수는 없다.
